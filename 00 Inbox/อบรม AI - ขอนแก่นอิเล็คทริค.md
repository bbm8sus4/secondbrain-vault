# อบรม AI — บ.ขอนแก่นอิเล็คทริค

## ภาพรวม
- **ลูกค้า:** บ.ขอนแก่นอิเล็คทริค (กลุ่ม LINE: ขอนแก่นอิเล็กทอร์นิค x [[EasySlip]])
- **ติดต่อผ่าน:** คุณป๊อบ (P-o-p) — เจ้าของ / ผู้ตัดสินใจ
- **ผู้เรียน:** 15 ท่าน — บัญชี, การตลาด, วิศวกร, ช่าง (บางคนเคยใช้ AI เบื้องต้น)
- **รูปแบบ:** 1 วัน (ยังไม่ล็อกวัน)

## หลักสูตรที่เจ้าของอยากได้ (แจ้ง 21 มิ.ย.)
1. พื้นฐาน Claude — ติดตั้ง สั่งงาน ฟังก์ชัน
2. ทำเว็บขึ้น production — GitHub, Vercel, Netlify
3. ฐานข้อมูล — Supabase, Firebase, MongoDB
4. GitHub
> "เทรนเหมือนเดิม" = มี curriculum มาตรฐานอยู่แล้ว

## ⚠️ ประเด็นต้องเคลียร์
- หลักสูตรสายเทคนิคลึก (deploy เว็บ + ฐานข้อมูล) **แต่ผู้เรียนคละแผนก** (บัญชี/การตลาด)
- ต้องถามเจ้าของ: คาดหวังให้ **ทุกคนทำได้** หรือ **เฉพาะทีมเทคนิค** → อาจต้องแบ่งกลุ่ม / เพิ่มวัน

## ต้นทุน (ยืนยันแล้ว)
- ห้องประชุม (รวม VAT): **฿3,745/วัน**
- เบรค + อาหาร: **฿290/หัว**
- → ต้นทุนปัจจุบัน **฿540/หัว** (ยังไม่รวมวิทยากร)
- วิทยากร 3 ท่าน สอนคนละเนื้อหา — **ยังไม่ระบุค่าตัว**
- ไม่มีค่าเดินทางวิทยากร

## Claude Max 5x (แยกจากต้นทุนคอร์ส — ผู้เรียนถือบัญชีเอง)
- ราคาจริง: $100/แอค/เดือน × ฿33.29 + ค่าธรรมเนียมบัตร 2.5% ≈ **฿3,412/แอค**
- ตารางแชร์ (15 คน):
  - 2 คน/แอค → 8 แอค = ฿1,820/หัว (ลื่นสุด)
  - **3 คน/แอค → 5 แอค = ฿1,137/หัว (แนะนำ — สมดุล)**
  - 4 คน/แอค → 4 แอค = ฿910/หัว (ตึง)
  - 5 คน/แอค → 3 แอค = ฿682/หัว (ตึงสุด = เท่า Pro/คน)
