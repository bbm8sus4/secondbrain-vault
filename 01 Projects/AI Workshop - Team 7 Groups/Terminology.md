# Terminology — คำที่ใช้ใน workshop นี้

## "ก้อนข้อมูลที่ส่งให้ ChatGPT"

ในวงการ AI prompting มีคำเรียกหลายแบบสำหรับก้อนข้อมูลที่เราป้อนให้ AI ก่อนสั่งงาน:

| คำ | ใช้ในวงการ | เหมาะแค่ไหน |
|---|---|---|
| **Business Context** | กว้าง · ทุกวงการ | โอเค แต่ไม่ครอบ "ผลงาน 4 คน" |
| **Brief** / **Brand Brief** | Agency · Marketing | คลาสสิก ฟังเหมือนเอกสารเริ่มงาน |
| **Context Pack** ⭐ | AI prompting modern | **ตรงสุด** · ครอบทั้ง profile + ผลงาน |
| **Briefing Bundle** | Workshop/training | เป็นทางการ ฟังดูเป็น deliverable |
| **Knowledge Pack** | RAG/Agent | เน้นว่าเป็น "ความรู้ที่ป้อนให้ AI" |
| **Source Pack** | Production/Creative | เน้นว่าเป็น "วัตถุดิบ" |

## ที่เลือกใช้ใน workshop

**"Context Pack"** — เพราะ:
- สั้น เข้าใจง่าย ผู้เรียนใหม่จับได้
- เป็นภาษาที่ใช้ใน AI prompting จริง (มี term "context window" / "context injection")
- ครอบได้ทั้งโปรไฟล์ + ผลงาน 4 คน
- พอเข้า Part 2 พูดว่า "วาง Context Pack → ยิงพรอมต์ Merge" ฟังลื่นเป็น flow

ภาษาไทย: **"ชุดบริบท"** (ใช้ได้ แต่ฟังแข็ง)

## คำอื่นใน workshop

| คำ | หมายถึง |
|---|---|
| **Project Instructions** | พรอมต์ที่ใส่ใน ChatGPT Project settings (system-level) — ใช้ทุกแชทใน project |
| **Sources** | ไฟล์ที่อัปโหลดใน ChatGPT Project (profile, logo, mood, product images) |
| **Code Interpreter** | Python sandbox ใน ChatGPT — ใช้ทำภาพ Proposal (text-heavy ไทย) |
| **DALL-E / GPT-4o image** | Image gen — ใช้ทำภาพคอนเทนต์ (visual) แต่อย่าใช้ใส่ตัวอักษรไทยยาว ๆ |
| **Persona / ICP** | Ideal Customer Profile + Persona (ลูกค้าในอุดมคติ + บุคลิกตัวแทน) |
| **Positioning** | จุดยืนของแบรนด์ในตลาด (เทียบคู่แข่ง) |
| **Value Proposition** | คุณค่าที่ลูกค้าได้ (ทำไมต้องเลือกแบรนด์นี้) |
| **Customer Journey** | เส้นทาง Awareness → Consideration → Purchase → Loyalty |
