SETUP · ทำครั้งเดียวก่อนเริ่ม

ตั้ง ChatGPT Project + ใส่ Project Instructions

ใน ChatGPT มีฟีเจอร์ "Projects" ที่ให้สร้างพื้นที่งานเฉพาะแบรนด์ — อัปโหลด Sources (profile + asset) ทีเดียว ใส่ Instructions ทีเดียว แล้วทุกแชทใน Project นั้นจะรู้บริบทอัตโนมัติ ไม่ต้อง paste context ซ้ำทุกครั้ง

ทำไมต้องเซ็ตอันนี้ก่อน: ถ้าไม่ตั้ง Project ผู้เรียนต้อง paste profile.md ทุกครั้งที่เปิดแชทใหม่ → เสียเวลา และมักลืม context สำคัญ → output ลอย ผิดแบรนด์
1

สร้าง Project ใหม่ใน ChatGPT

กด "+ New Project" ที่ sidebar ซ้าย → ตั้งชื่อเป็นชื่อบริษัทของทีม (เช่น "GreenLeaf Furniture", "Bloom & Co.") — ไม่ต้องใส่ emoji

2

Add Sources — อัปโหลดเอกสารและรูปทั้งหมด

เปิดแท็บ Sources → "Add sources" → อัปโหลด:

  • profile.md (บังคับ)
  • รูปทุกใบในโฟลเดอร์ Logo/, CI Branding/, Mood Board/
  • รูปสินค้าทั้ง 6 ใบในโฟลเดอร์ Product/
  • ไฟล์เพิ่มเติมที่ทีมสร้างเอง (เช่น brand strategy, marketing matrix) ก็ใส่ได้
3

เปิด Project settings → ช่อง Instructions

กดจุดสามจุดมุมขวาบนของ Project → Project settings → จะเจอช่อง Instructions: Set context and customize how ChatGPT responds in this project

4

วาง Project Instructions ด้านล่าง — แล้ว Save

พรอมต์กลางต่อไปนี้ใช้ได้กับทุกบริษัท ทุกกลุ่ม — ไม่ต้องแก้ ChatGPT จะดึงบริบทแบรนด์จาก Sources ที่อัปโหลดเอง

PROJECT INSTRUCTIONS — กลาง ใช้ได้กับทุกธุรกิจ วางใน Project settings 
คุณคือผู้ช่วยทำงานให้ทีมการตลาดของแบรนด์นี้
เพื่อสร้าง Brand & Go-to-Market Proposal และคอนเทนต์ของแบรนด์

[บริบท]
รายละเอียดของแบรนด์ (ชื่อบริษัท ธุรกิจ ที่ตั้ง ขนาด USP กลุ่มลูกค้า
ช่องทางขาย วิสัยทัศน์ คู่แข่ง ฯลฯ) อยู่ใน Sources ที่อัปโหลดไว้ใน Project นี้
เช่น profile.md และไฟล์อื่น ๆ ของแบรนด์ (logo, mood board, รูปสินค้า)

ทุกคำตอบต้องอ้างอิงข้อมูลใน Sources เป็นหลัก ไม่ใช่ความรู้ทั่วไป
ก่อนเริ่มงานชิ้นแรกในแชท ให้สรุปสั้น ๆ ว่า "เข้าใจแบรนด์นี้ว่าคือ..."
แล้วค่อยทำงานต่อ — เพื่อให้ผู้ใช้ตรวจว่าคุณดึงข้อมูลถูกต้อง

[หน้าที่]
ในแต่ละแชท ผู้ใช้จะระบุว่ารับบทไหนใน 4 บทบาท:
- Strategist (นักการตลาด) → Positioning · Brand Identity · Messaging · Value Proposition
- Researcher (นักวิจัยตลาด) → ICP/Persona · Customer Journey · Pain Point · Buying Behavior
- Sales (ผู้ช่วยฝ่ายขาย) → Pricing · Promotion · Sales Forecast · Channel
- Content (นักเขียนคอนเทนต์) → Communication · Content Plan · Campaign

ตอบเฉพาะมุมของบทบาทนั้น แต่ดึงข้อมูลร่วมจาก Sources ทุกครั้ง
ถ้าผู้ใช้ไม่ระบุบทบาท ให้ถามก่อนว่ารับบทไหน

[กติกาตอบ]
- ภาษาไทยธรรมชาติ เหมือนคนไทยคุยกันจริง ไม่ใช่ภาษาแปลหรือราชการ
- กระชับ ตรงประเด็น ขายได้จริง ไม่อ้อมค้อม
- ทุกข้อสรุปต้องเชื่อมกับ "ตัวตนของแบรนด์นี้" เสมอ ไม่ใช่ template ลอย ๆ
- ถ้ามีตัวเลข ให้บอกฐานคำนวณ/สมมุติฐานควบคู่
- ถ้าผู้ใช้ขอสร้างภาพที่มีตัวอักษรไทยเยอะ ให้ใช้ Code Interpreter
  (Python + matplotlib หรือ PIL) ไม่ใช้ DALL-E เพราะตัวอักษรไทยจะเพี้ยน
- ถ้าสร้างภาพคอนเทนต์ (ไม่ใส่ text) ใช้ภาพอ้างอิงจาก Sources เป็นหลัก
  (รูปสินค้า · mood board · logo) เพื่อให้สี/สไตล์ตรงแบรนด์

[ห้าม]
- ห้ามใช้ emoji เยอะ หรือสำนวน clickbait แบบ "Sale!! ลดสุดสุด!!"
- ห้ามใช้ภาษาราชการ ("ดำเนินการเรียบร้อย" "กรุณาตรวจสอบ")
- ห้ามแนะนำให้ผู้ใช้ "ไปค้นในเว็บ" — ให้ตอบจาก Sources ที่มี
- ห้าม fabricate ตัวเลข — ถ้าไม่มีใน Sources ให้ระบุว่า "ประมาณการ"
  พร้อมแจ้งสมมุติฐาน
- ห้ามเปลี่ยนตัวตน/ตำแหน่งของแบรนด์ระหว่างแชท ถ้าผู้ใช้ขอแบบที่ขัดกับ
  Positioning ใน Sources ให้ทักก่อน
หลังตั้งเสร็จ — ทดสอบ 1 ครั้ง: เปิดแชทใหม่ใน Project พิมพ์ "สรุปแบรนด์นี้ให้ฟัง 3 บรรทัด" → ถ้า ChatGPT สรุปได้ตรงกับ profile.md = setup ผ่าน พร้อมเข้า Part 1 ถ้าสรุปลอย หรือไม่ตรง = ตรวจว่าอัปโหลด Sources ครบหรือยัง

PART 1 · ตัวคนเดียว

8 สเตปแต่ละคนทำงานตามบทบาท

ก่อนรวมทีม แต่ละคนต้องได้ output ของบทบาทตัวเองออกมาก่อน — ทำสเตปต่อสเตปไม่ข้าม

1

รู้ตัวเอง 3 อย่าง

ก่อนแตะ ChatGPT — เคลียร์ในใจให้ได้ว่า ตัวเองอยู่กลุ่มไหน บริษัทอะไร บทบาทอะไร เขียนแปะกระดาษไว้กันลืม

2

เปิด 2 ของพร้อมกัน

1) โฟลเดอร์บริษัทตัวเอง — อ่าน profile.md ให้จบ + ดูรูปสินค้า/โลโก้/Mood
2) AI Prompt Playbook — เปิด ชีตของบทบาทตัวเอง ชีตเดียวพอ ไม่ต้องดูของคนอื่น

3

หาพรอมต์ที่ตรงงานในชีต

ดูคอลัมน์ "สถานการณ์ใช้งาน" เลือกที่ตรงกับสิ่งที่บทบาทตัวเองต้องส่ง:

  • Strategist — Positioning · Brand Identity · Messaging · Value Proposition
  • Researcher — ICP/Persona · Customer Journey · Pain Point · Behavior
  • Sales — Pricing · Promotion · Forecast · Channel
  • Content — Content Plan · Campaign · Communication
4

เปิด ChatGPT แชทใหม่ + paste พรอมต์

ขึ้นแชทใหม่เสมอสำหรับงานใหม่ ไม่ปนกับเรื่องอื่น แล้ว copy พรอมต์จากชีตวางลงไป

5

แทน {{ตัวแปร}} ทุกตัว

หาช่อง {{ }} ในพรอมต์ → ดึงข้อมูลจาก profile.md มาใส่

ตัวอย่าง: {{brand_name}} → "GreenLeaf Furniture" · {{target}} → คัดมาจากหัวข้อ Target Customers ใน profile
6

รัน ChatGPT แล้วอ่านอย่างจริงจัง

ถ้า output ลอย/ทั่วไปเกินไป → บอก ChatGPT ปรับ เช่น "เจาะจงกับบริษัทฉันมากกว่านี้" · "สั้นลง 50%" · "เป็นภาษาที่ลูกค้าเข้าใจ" · "เพิ่ม 2 ตัวอย่าง"

7

เก็บผลลัพธ์ลง Google Doc ส่วนตัว

ก่อนทำหัวข้อต่อไป — copy ผลลัพธ์ที่พอใจเก็บไว้ใน Doc ของบทบาทตัวเอง ไม่ต้องรอเสร็จทั้งหมด

8

ทำซ้ำ 3-6 จนครบหัวข้อของบทบาทตัวเอง

เปลี่ยนพรอมต์ในชีตเป็นข้อถัดไป แล้ววนทำใหม่ จนได้ output ครบทุกหัวข้อย่อยของบทบาท

เคล็ดที่ห้ามลืม: ห้ามรัน ChatGPT แบบไม่อ่าน profile ก่อน — output จะลอย ถ้าใส่ context จาก profile.md เยอะ ผลจะคม เพราะพรอมต์ในชีตออกแบบให้เติมตัวแปรเข้าไป ไม่ใช่ใช้ดิบ ๆ
PART 2 · รวมทีม

รวม output 4 คน เป็นไฟล์สรุปก้อนเดียว

เมื่อ 4 คนได้ผลลัพธ์ของตัวเองครบแล้ว — มารวมกันใน ChatGPT แชทเดียวด้วย 3 พรอมต์ ก่อนที่จะเอาไปทำภาพ Proposal

1) เปิดแชท ChatGPT ใหม่ (ของทีม) 2) แปะ context — profile บริษัท + ผลงาน 4 บทบาท 3) ยิงพรอมต์ "Merge & Structure" → ได้ draft ครบ 15 หัวข้อ 4) ยิงพรอมต์ "Tighten" → ตัดสั้นพอลงในภาพ 5) ยิงพรอมต์ "Self-check" → เจอจุดอ่อน + แก้ 6) Copy ผลลัพธ์ลง Google Doc กลาง = ไฟล์สรุปทีม

พรอมต์ 1 · Merge & Structure

รวบผลงานทุกคน → เรียงตาม 3 บล็อก 15 หัวข้อ ตามที่ workshop กำหนด

PROMPT 1 — รวบเป็นโครงสร้างเดียว งานหลัก 
ทำหน้าที่เป็น Marketing Director ที่ต้องส่งงาน Brand & Go-to-Market Proposal
ให้กับทีมผู้บริหาร

ภารกิจ: นำผลงาน 4 บทบาทด้านบนมาประกอบเป็นเอกสารสรุป 1 ฉบับ
ตามโครงสร้าง 3 บล็อก 15 หัวข้อย่อย "เป๊ะตามนี้":

[บล็อก 1] BRAND STRATEGY (แบรนด์ที่ชัดและต่าง)
1) Positioning — แบรนด์ยืนอยู่จุดไหนในตลาด
2) Brand Identity — อัตลักษณ์ (โทน เสียง บุคลิก)
3) Brand Messaging — สารหลักที่จะสื่อสาร
4) Value Proposition — คุณค่าที่ลูกค้าได้

[บล็อก 2] TARGET CUSTOMER (รู้ว่าลูกค้าคือใคร)
5) ICP / Persona — ลูกค้าในอุดมคติ 1 persona พร้อมชื่อ อายุ อาชีพ พฤติกรรม
6) Customer Journey — Awareness → Consideration → Purchase → Loyalty
7) Pain Point — ปัญหา 3 ข้อหลัก
8) Buying Behavior — ซื้อเมื่อไหร่ ที่ไหน อย่างไร

[บล็อก 3] SALES & MARKETING PLAN (สร้าง Awareness → Lead → ยอดขาย)
9) Communication Strategy — สื่อสารยังไง ใช้ tone ไหน
10) Channel Strategy — เน้นช่องไหน เพราะอะไร
11) Content Plan — คอนเทนต์หลัก 3-5 ธีม
12) Campaign Plan — แคมเปญใหญ่ 1 ตัว พร้อม objective + KPI
13) Pricing Strategy — ราคา/แพ็คเกจ
14) Promotion Strategy — โปรโมชั่นช่วงเปิดตัว
15) Sales Forecast — คาดการณ์ยอดขาย 3 เดือนแรก

PART 3 · ภาพ Proposal

สร้างภาพ Proposal ด้วย Code Interpreter

ใน ChatGPT มี 2 เครื่องมือสร้างภาพ — สำหรับภาพ Proposal ที่มีตัวอักษรเยอะ 15 หัวข้อ ต้องใช้ Code Interpreter (Python) เพราะ DALL-E เขียนภาษาไทยยาว ๆ ยังไม่เป๊ะ

ห้ามทำเด็ดขาด: สั่ง DALL-E ทำภาพ Proposal โดยตรง → ตัวอักษรไทยจะเพี้ยน อ่านไม่ออก ผู้เรียนจะเสียเวลา
1

เปิด Code Interpreter

ใน ChatGPT พิมพ์ / แล้วเลือก "Run Python" หรือคลิกไอคอนคลิป → "Analyze data" จะเข้า Python sandbox

แปะผลสรุปสุดท้ายจาก Part 2

วาง 15 หัวข้อย่อยที่ผ่าน 3 พรอมต์มาแล้ว — ChatGPT จะเอาไปเป็นข้อมูลในภาพ

3

ยิงพรอมต์ "วาดภาพ Proposal"

พรอมต์ด้านล่าง copy ใช้ได้เลย — แก้แค่ชื่อบริษัท + แปะ output 15 หัวข้อ

4

Iterate ในแชทเดียว ไม่สั่งใหม่ทั้งภาพ
