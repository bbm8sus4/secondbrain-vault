# 1 · คอนเทนต์โซเชียล
## ให้ Claude เป็น copywriter ประจำทีม

---

## จาก 1 ไอเดีย → คอนเทนต์หลายชิ้น

จุดแข็งของ Claude คือ **ขยายไอเดียเดียวเป็นหลายมุม** ได้เร็ว

```
เรามีโปรโมชัน: เครื่องซักผ้าลด 30% ถึง 15 ก.ค.

ช่วยเขียนโพสต์ Facebook 3 แบบ มุมต่างกัน:
1) เน้นประหยัดเงิน
2) เน้นแก้ปัญหา (ซักผ้าไม่ทัน)
3) เน้นความเร่งด่วน (ใกล้หมดเวลา)

แต่ละอัน 2-3 บรรทัด ปิดด้วย CTA ทัก LINE ไม่ต้องมี emoji
```

> **ได้ 3 โพสต์ในคำสั่งเดียว** — เลือกอันที่ใช่ หรือผสมกัน

---

## ปรับ 1 โพสต์ให้เข้าทุกแพลตฟอร์ม

โพสต์เดียวกัน แต่ละที่ต้องการคนละแบบ — สั่ง Claude ปรับให้

| แพลตฟอร์ม | สไตล์ที่ Claude ปรับให้ |
|---|---|
| **Facebook** | เล่าเรื่องได้ ยาวหน่อยได้ |
| **LINE / broadcast** | สั้น ตรง มีปุ่ม/ลิงก์ชัด |
| **TikTok / Reels** | สคริปต์พูด + hook 3 วินาทีแรก |

```
เอาโพสต์นี้ ปรับเป็น 3 เวอร์ชัน: Facebook, ข้อความ LINE สั้น,
และสคริปต์คลิป TikTok 15 วินาที
```

> **สำคัญ:** งานที่เคยทำครึ่งวัน เหลือ 10 นาที — เวลาที่เหลือเอาไปตรวจ+ปรับให้ดี

---

## ลองเลย 1 — คอนเทนต์ของคุณเอง (10 นาที)

**โจทย์:** เลือกโปรโมชัน/สินค้าจริงของบริษัท 1 อย่าง

<div class="step">

**1)** บอก Claude ว่าขายอะไร · ลูกค้าเป็นใคร · อยากได้กี่แบบ

</div>

<div class="step">

**2)** ใส่ **ตัวอย่างโพสต์เก่าที่เคยเวิร์ก** (เทคนิคจากเช้า)

</div>

<div class="step">

**3)** ขอ 3 เวอร์ชัน แล้วเลือก/ผสมอันที่ชอบ

</div>

<div class="ok">

**เป้าหมาย:** ได้คอนเทนต์จริง 1 ชุด ที่โพสต์ได้เลย · ต่อ/นัท เดินช่วยใครติด

</div>

# 2 · งานภาพ
## Claude ช่วยเรื่องภาพยังไง

---

## เข้าใจก่อน — Claude ไม่ได้ "วาดรูป" ให้

<div class="warn">

Claude เก่ง **ภาษาและความคิด** — ไม่ได้ generate รูปถ่าย/ภาพวาดเอง

</div>

แต่ช่วยงานภาพได้ **2 ทาง**:

- **เขียนบรีฟ/prompt ภาพ** ให้ละเอียด → เอาไปเข้าเครื่องมือสร้างภาพ AI
- **ทำ visual ด้วย Artifacts** → โปสเตอร์ · การ์ด · ตาราง · infographic แบบ HTML

> **คิดง่ายๆ:** Claude = "ผู้กำกับศิลป์" ที่สั่งงานเก่ง ไม่ใช่ "คนวาด"

---

## ทาง A — ให้ Claude เขียนบรีฟภาพ

แทนที่จะนึก prompt ภาพเอง ให้ Claude เขียนให้ละเอียด

```
ฉันจะทำภาพโปรโมชันเครื่องซักผ้าลงเฟซ
ช่วยเขียน prompt ภาษาอังกฤษให้ละเอียด สำหรับเครื่องมือสร้างภาพ AI:
- โทนสะอาด น่าเชื่อถือ สีน้ำเงิน-ขาว
- มีเครื่องซักผ้า + ป้ายลด 30%
- พื้นที่ว่างซ้ายบนไว้ใส่ข้อความ
```

> **ได้ prompt พร้อมใช้** → เอาไปวางในเครื่องมือสร้างภาพที่บริษัทใช้

---

## ทาง B — ทำ visual ด้วย Artifacts

สั่ง Claude ทำชิ้นงานภาพแบบ **HTML** ที่เห็น preview สดๆ

- **การ์ดโปรโมชัน** — ข้อความ + layout จัดสวย
- **ตารางเปรียบเทียบสินค้า** — ให้ลูกค้าเห็นชัด
- **infographic โครงง่ายๆ** — 3-4 ขั้นตอน / สถิติ

```
ทำการ์ดโปรโมชันขนาดโพสต์ IG (สี่เหลี่ยมจัตุรัส)
หัวข้อ "ลด 30%" · สินค้าเครื่องซักผ้า · ปุ่มทัก LINE
โทนน้ำเงิน-ขาว ดูพรีเมียม
```

> **Artifacts** = เห็นผลทันทีด้านขวา · แก้จนพอใจ · แล้ว capture/export

# 3 · คลัง Prompt ของทีม
## เก็บของดีไว้ใช้ซ้ำ

---

## ทำไมต้องมีคลัง prompt

ทุกครั้งที่เจอ prompt ที่ **ได้ผลดี** — อย่าปล่อยหาย

<div class="ok">

- ครั้งหน้าไม่ต้องคิดใหม่ · copy มาปรับนิดเดียว
- คนใหม่เข้าทีม เอาไปใช้ได้เลย
- ทั้งทีมเก่งขึ้นพร้อมกัน ไม่ใช่เก่งคนเดียว

</div>

> **นี่คือของที่จะได้กลับไปใช้ที่ออฟฟิศจริง** — ไม่ใช่แค่ความรู้ในหัว

---

## เก็บ prompt แบบไหนให้ใช้ต่อได้

เก็บ 3 อย่างต่อ 1 prompt:

| เก็บอะไร | ตัวอย่าง |
|---|---|
| **ชื่องาน** | "เขียน caption FB โปรโมชัน" |
| **ตัว prompt** | คำสั่งเต็มที่ใช้ (พร้อมช่อง [แก้ตรงนี้]) |
| **ตัวอย่างผลลัพธ์** | โพสต์ที่ออกมาแล้วเวิร์ก |

> **ที่เก็บ:** Google Doc / Sheet กลางของทีม 1 ไฟล์ — ใครเจอ prompt ดี เพิ่มเข้าไป

---

## ลองเลย 2 — เริ่มคลังทีมวันนี้ (5 นาที)

**โจทย์:** เอา prompt ที่เพิ่งใช้ในคาบนี้ ลงคลังกลาง 1 อัน

<div class="step">

**1)** เปิด Doc กลางของทีม (ต่อ/นัท เตรียมลิงก์ไว้)

</div>

<div class="step">

**2)** วาง 3 อย่าง: ชื่องาน + prompt + ตัวอย่างที่ได้

</div>

<div class="ok">

**เป้าหมายทีม:** จบ workshop มี prompt ในคลังอย่างน้อย 10 อัน — เริ่มจากวันนี้

</div>

---

## Recap — จำ 4 อย่างพอ

1. **1 ไอเดีย → หลายโพสต์** สั่ง Claude ขยายมุม + ปรับตามแพลตฟอร์ม
2. **Claude ไม่วาดรูป** แต่เขียนบรีฟภาพเก่ง + ทำ visual ด้วย Artifacts
3. **เก็บ prompt ที่เวิร์ก** ลงคลังกลาง — ชื่องาน + prompt + ตัวอย่าง
4. **AI ร่างให้ เราตรวจ+ปรับ** เสมอ ก่อนเอาออกจริง
