# Menu Catering — SHIFT

**ร้าน:** SHIFT (ขนมปัง + เค้ก + อาหารกล่อง)
**วันที่กำหนด:** 2026-06-30 (รายการที่ใช้)
**ใช้ที่:** ฟอร์มผู้เรียน ส่วนที่ 7 (item #32-35)

## เรทราคา

### อาหารกลางวัน — 190.-/เซต

1 ชุดประกอบด้วย: กับข้าว 2 อย่าง + ข้าวเปล่า + ผลไม้ + น้ำผลไม้ 1 กล่อง + น้ำเปล่า 1 ขวด

15 เมนู:
- ข้าวกะเพราหมู / ไก่
- ข้าวผัดพริกหมู / ไก่
- ข้าวแกงจืดหมู / ไก่
- ข้าวผัดหมู / ไก่ / ไส้กรอก
- ไข่เจียวหมูสับ / ไก่สับ
- ข้าวหมูเกาหลี / ไก่เกาหลี
- ข้าวหมูสับผัดซีอิ๊ว / ไก่สับผัดซีอิ๊ว

### เครื่องดื่มเย็นเบรค — 70.-/แก้ว

เฉพาะจัดเบรค:
- Americano (เย็น)
- Iced Milk
- Matcha Pure
- Matcha Latte
- Thai Tea
- Lychee Soda

### Break Set — 49.-/ชุด (เลือก "ขนม")

= ขนม 1 ชิ้น (ราคา 18 / 20 / 25.-) + นม/น้ำผลไม้ 1 กล่อง

> Break Set ราคาต่ำกว่า (39.-) มีให้เลือก = ขนมราคา 18.- เท่านั้น + นม/น้ำผลไม้
> **โปรเจกต์นี้ใช้ Break Set 49.-** เพื่อให้เลือกขนมได้ทุกราคา
