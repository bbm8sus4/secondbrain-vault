## กำไรจริงดูตรงไหน

- **กำไรรวม** (กล่องเขียว) = กำไรจริง **ก่อนภาษีเงินได้** = รายได้ − ต้นทุนทั้งหมด
- **เงินโอนเข้าจริง** (กล่องล่าง) = เงินสดที่ลูกค้าโอน = รายได้ + VAT − หัก ณ ที่จ่าย → **ไม่ใช่กำไร** (ยังไม่หักต้นทุน)
- VAT + หัก ณ ที่จ่าย **ไม่กระทบกำไรทางบัญชี** (VAT นำส่งสรรพากร · WHT เป็นเครดิตภาษีได้คืน) — มีผลแค่กระแสเงินสด

ตัวอย่างแผน D (15 คน × ฿2,499): ต้นทุน/หัว 1,463 · กำไร/หัว 1,036 · กำไรรวม **15,545** · margin 41.5%

## Logic ภาษี (สำคัญ — อย่าสับสน)

แยกเป็น 2 มิติ ที่คิดคนละฐาน:

| มิติ | ดูที่ | กติกา |
|---|---|---|
| **VAT** | ผู้ขาย | นิติบุคคลจด VAT → บวก 7% · บุคคลธรรมดาปกติไม่จด (รายได้ < 1.8 ล้าน/ปี) → ไม่บวก |
| **หัก ณ ที่จ่าย** | ผู้จ่าย | ม.3 เตรส — หักเฉพาะ**ผู้จ่ายที่เป็นนิติบุคคล** · บุคคลธรรมดาจ่าย → ไม่มีหน้าที่หัก (0%) |

**อัตราหัก ณ ที่จ่าย คิดตามประเภทเงินได้ ไม่ใช่ตามนิติ/บุคคล:**
- **3%** = ค่าบริการ / จ้างทำของ / วิทยากร-อบรม / ฟรีแลนซ์ / ที่ปรึกษา (service & professional fees)
- **5%** = ค่าเช่าห้อง / พื้นที่ / ทรัพย์สิน (rents)
- งาน AI Workshop = **ค่าบริการอบรม → หัก 3%** (ไม่ใช่ 5%) — ตรงกับใบเสนอราคา KETE: 37,485 × 3% = 1,124.55 → โอนสุทธิ 36,360.45

## 3 โหมดคู่ค้า

| โหมด | VAT | หัก ณ ที่จ่าย | เงินโอนจริง (จาก 37,485) |
|---|---|---|---|
| 🏢 นิติ → นิติ | มีได้ 7% | 3% (บริการ) / 5% (เช่า) | 36,360 (VAT off) |
| 👤 บุคคล → นิติ | ไม่มี | 3% / 5% | 36,360 |
| 👤 บุคคล → บุคคล | ไม่มี | **ไม่มี (0%)** | 37,485 |

โหมดบุคคล→บุคคล: ปุ่มเลือกประเภทเงินได้ถูกปิดอัตโนมัติ (ไม่มีการหัก) · ทุกโหมดยังไม่รวมภาษีเงินได้บุคคลธรรมดาของผู้ขาย
