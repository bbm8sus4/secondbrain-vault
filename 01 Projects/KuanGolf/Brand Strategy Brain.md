## 1) Business Brain

### ธุรกิจนี้คืออะไร
- **Surface**: Sports SaaS B2C — Golf scoring + side-bet calculator (PWA + LINE-native), ฿999-2,490
- **ของจริง**: เครื่องมือ **ลด social blame** ที่หัวก๊วนแบกรับทุกรอบ ในรูป SaaS subscription
- **Stage**: pre-traction (v0.1.0, 0 user, 0 paying, FB 2 posts)
- **Unfair infra**: embedded ใน Thunder Solution ecosystem — EasySlip + BoostSMS + EasyCRM ใช้ได้ทันที
- **Revenue path**: B2C subscription (เดือน 1-12) → B2B/B2B2C สนาม (เดือน 6+) → Inbound golfer EN/JP/KR (เดือน 12+)

### ลูกค้าหลัก
- **"Captain หัวก๊วน"** — บทบาทเฉพาะในก๊วน ไม่ใช่ "golfer" ทั่วไป
- **ภาระเฉพาะ**: คนเดียวที่จัดสนาม + คิดเงิน + เก็บเงิน + รับ blame ถ้าผิด
- **Demo**: ชาย 45-65, SME owner/executive, A/B+, Bangkok metro + ปริมณฑล
- **Pricing power**: greenfee ฿1.5-7.5K/รอบ × 2-4 รอบ/เดือน → ฿999/ปี ไม่ใช่ปัญหา
- **Multiplier**: 1 Captain → 4-12 คนในก๊วน → CAC ต่ำ organic

### ตลาดต้องการอะไร (3-layer demand)

| Layer | Demand | สถานะ supply |
|---|---|---|
| 1 | GPS, scorecard, handicap, coaching | **อิ่มตัว** (18Birdies, Garmin, BirdieX, TGA) |
| 2 | Side-bet calculator ไทย + game format ไทย | **underserve** (โลกัณฑ์ฯ พอใช้, Beezer ดีแต่ไม่ไทย) |
| 3 | **PromptPay settlement + LINE summary + ตัดบทบาทเหรัญญิก** | **0 competitor** |

→ ห้ามแข่ง Layer 1 (แพ้แน่) ต้องครอง Layer 3 ก่อน
**TAM**: นักกอล์ฟไทยประจำ 500K + inbound 700K-1.1M/ปี → Captain ~10-20% = **50-100K serviceable** (เล็กแต่ ARPU สูง)

### คู่แข่งขายอะไร

| ค่าย | ขายอะไร | ของจริงที่ลูกค้าจ่าย |
|---|---|---|
| กระดาษ + LINE chat | ฟรี + ความเคยชิน | "ไม่ต้องเรียนรู้ใหม่" |
| โลกัณฑ์กอล์ฟ | calculator ไทย | "ตรงไปตรงมา + ภาษาไทย" |
| Beezer Golf | 28 games + BEEZER Bank | "variety + ledger ข้ามรอบ" |
| BirdieX/golfdigg | social + booking ecosystem | "ครบ end-to-end" |
| 18Birdies/Hole19 | GPS + handicap + brand | "premium global" |
| SmartScore | B2B tablet + course infra | "smart course experience" |
| Garmin | watch + standalone | "ไม่ต้องใช้มือถือ" |
| DFG | scoring ไทย (UX เก่า) | "first mover ไทย" |

**Insight**: ไม่มีใครขาย "ตัดบทบาทเหรัญญิก + PromptPay + LINE summary" pure play = unowned territory

### จุดแข็งแบรนด์ (5 strength)
1. **Thunder Solution infra** — EasySlip + BoostSMS + EasyCRM → ทำ PromptPay flow ได้เร็ว+ถูก = สิ่งที่ Beezer/BirdieX ทำไม่ได้
2. **Bob = COO** — product จะ practical ไม่ใช่ engineering toy
3. **Blank slate** — 0 user เก่า = 0 constraint, pivot เร็ว
4. **Specialist DNA** — niche แคบ + lean
5. **Native Thai stack** — ภาษา + วัฒนธรรม + PromptPay + LINE

### จุดอ่อนที่ต้องยอมรับ (5)
1. pre-traction — 0 user, 0 paying
2. 0 validated insight — สมมติฐานล้วน
3. ไม่มี media engine (golfdigg/InVent มี)
4. ไม่มี course distribution (SmartScore/BirdieX มี)
5. มาทีหลัง 1-3 ปี (Beezer มี Bank, BirdieX มี InVent, DFG มี 4,500 users)

### โอกาสในตลาด (priority sorted)
1. 🔥 **PromptPay settlement (Layer 3)** — 0 competitor, Thunder infra ทำได้ → เร่งสุด
2. 🔥 **Captain network multiplier** — 1 → 4-12 organic, switching cost สูง
3. 🟠 **Window 12 เดือน** — ก่อน BirdieX/InVent push + Beezer ขยาย SEA + 18Birdies localize
4. 🟠 **TGA partnership** — official handicap = structural moat
5. 🟢 **B2B B2B2C สนาม** (phase 2) — ชนะ Modern Golf System (UI เก่า)
6. 🟢 **Inbound golfer EN/JP/KR** (phase 3) — TAM 700K-1.1M

### สรุป Brain ในประโยคเดียว
> KuanGolf ไม่ใช่ "scoring app อีกตัว" — เป็นทางออกของหัวก๊วน 50-100K คนในไทยที่ไม่อยากเป็นเหรัญญิก โดยใช้ PromptPay + LINE + Thunder Solution stack ที่ไม่มีคู่แข่งทำได้
> **Window**: 12 เดือน · **Moat**: 4 มุม converge · **Risk**: Beezer ขยาย SEA หรือ TGA ออก app ทางการ

## 2) Brand Value

### คุณค่าที่ลูกค้าได้รับจริง (Functional)
- คำนวณเสร็จใน 30 วินาที — กดสกอร์ แอปคิดให้หมด
- QR PromptPay ส่งทุกคนใน LINE ก๊วน — แตะแล้วโอนเลย
- มีหลักฐานทุกบาท — ย้อนดูได้
- เข้าใจเกมไทย — ต่อแต้ม, halve-on-tie, System 36

### คุณค่าทางอารมณ์ (Emotional)
- เลิกเป็นจำเลย — คนคำนวณไม่ใช่คนถูกสงสัย
- กลับมาสนุกกับรอบ — หัวก๊วนกลับมาเป็นคนเล่น ไม่ใช่เหรัญญิก
- จบทะเลาะก่อนเกิด — มีหลักฐานชัด
- ภูมิใจว่าก๊วนเราจัดการเป็น

### คุณค่าที่ทำให้ต่างจากคู่แข่ง (Differentiation)
- อยู่ใน LINE ที่ก๊วนใช้อยู่แล้ว — ไม่ต้องลากเพื่อน 8 คนโหลด app
- PromptPay ตรงๆ — ไม่ใช่ wallet ไม่ใช่บัตร ไม่ใช่เงินค้าง
- ทำเรื่องเดียวให้ดีที่สุด — ไม่ใช่ "แอปกอล์ฟครบทุกอย่าง"
- เกิดมาเพื่อหัวก๊วน — ไม่ใช่ scoring tool ที่ต้องดัดแปลงใช้

### คุณค่าที่แบรนด์ยืนหยัด (Brand Stance)
- เป็นกลาง — แอปไม่เข้าข้างใคร
- โปร่งใส — ทุกบาทมีที่มา
- เคารพวิธีเล่นแบบไทย — ของก๊วนไทย
- จบในรอบ — ไม่ลาก ไม่ค้าง

### 🎯 3 Brand Value หลักที่ใช้สื่อสาร
1. **"ไม่มีใครเป็นจำเลย"** — หัวใจแบรนด์ (Captain blame avoidance), ไม่มีคู่แข่งพูด
2. **"จบในรอบ ส่งเข้า LINE"** — ความเร็ว + workflow ที่คนไทยใช้อยู่แล้ว
3. **"ของก๊วนไทย ไม่ใช่ตำราฝรั่ง"** — cultural moat (ภาษา + PromptPay + วัฒนธรรม)

## 3) Brand Positioning

### เราควรยืนตรงไหน
**"Settlement specialist สำหรับก๊วนไทย"**
- ❌ ไม่ยืนใน feature war / ecosystem war / hardware war (แพ้หมด)
- ✅ ยืนที่ Layer 3 demand — settlement + LINE + PromptPay
- ✅ ยืนข้างกระดาษ + LINE chat แล้วทำให้ฉลาดขึ้น
- ✅ ยืนเป็น "ตัวกลางเป็นกลาง" ไม่ใช่ "scoring tool ของ golfer"

### ต่างจากคู่แข่งอย่างไร

| มิติ | คู่แข่ง | KuanGolf |
|---|---|---|
| User identity | Golfer ทั่วไป | **หัวก๊วน** specific role |
| โฟกัส | ครบทุกอย่าง | **เรื่องเดียว** — settlement |
| Workflow | เปิด app เอง | **LINE-native** |
| เงิน | บัตร / wallet / commission | **PromptPay QR คน-ต่อ-คน** |
| เกม | global (USGA) | **ไทยแท้** — ต่อแต้ม, halve-on-tie |
| ขนาดก๊วน | Match 72-180 คน | **ก๊วน 4-12 คน** |
| อารมณ์ | "เล่นกอล์ฟดีขึ้น" | **"ไม่ต้องเป็นจำเลย"** |

### ลูกค้าควรจำเราเรื่องอะไร
> **"หัวก๊วน ไม่ต้องเป็นเหรัญญิก"** — จำได้แค่ 1 เรื่องเท่านั้น

### Positioning Statement
**External (1 ประโยค)**:
> "KuanGolf — หัวก๊วนไทยที่ฉลาด ไม่ต้องเป็นเหรัญญิกอีกต่อไป จบในรอบ ส่ง QR + สรุปเข้า LINE ก๊วน"

**Internal (สำหรับทีม)**:
> สำหรับหัวก๊วนไทย 45-65 ที่จัดก๊วน 4-12 คน เล่น 2-4 รอบ/เดือน และเบื่อกับการเป็นคนคิดเงินและรับ blame — KuanGolf คือ settlement specialist ที่ลบบทบาทเหรัญญิกออกจากหัวก๊วน โดยคำนวณ + ส่ง QR PromptPay + สรุปเข้า LINE ใน 30 วินาที ต่างจาก scoring apps อื่นที่บังคับให้หัวก๊วนคิดเงินเอง เพราะเกิดมาเพื่อหัวก๊วนโดยเฉพาะ บน Thunder Solution stack ที่คนไทยใช้อยู่จริง

### คำที่ควรใช้
- **identity**: หัวก๊วน, ก๊วน, เพื่อนก๊วน, เพื่อนซี้
- **pain**: เหรัญญิก, จำเลย, "ไว้คุยกันทีหลัง", ดราม่าหลังรอบ
- **value**: จบในรอบ, 30 วินาที, เป็นกลาง, โปร่งใส, มีหลักฐาน, ส่งเข้า LINE, QR PromptPay
- **game**: ต่อ, ต่อแต้ม, halve-on-tie, System 36, รอบนี้/รอบหน้า

### คำที่ห้ามใช้
- **ทำให้เหมือนคู่แข่ง**: Score tracking app, GPS, Handicap tracker, All-in-one, ครบครัน, Tournament, Society, Leaderboard, Social golf experience, AI-powered
- **legal risk**: Wallet, บัญชี, เงินในแอป, เดิมพัน, พนัน, betting, Bank
- **ภาษา tech**: Subscription → "ปีนึงเลย", Onboard → "เริ่มใช้", Dashboard → "หน้าหลัก"
- **โทนผิด**: ตลก/ทะลึ่ง, self-deprecating, hyperbole ("ปฏิวัติวงการ")

### โทนแบรนด์
- เหมือนเพื่อน Captain ที่อาวุโสกว่า 5 ปี — เข้าใจ, ตรง, ไม่อ้อม
- ภาษาคน 100% · มั่นใจ ไม่ขายดี · เคารพลูกค้า · สั้น ตัด

## 4) Customer Persona

### "พี่เอก" (Captain หัวก๊วน)
> ข้อมูล demographic/geo/behavior = จริง · emotional/buying logic = **[สมมติฐาน]** ต้อง validate
> Tagline ในใจ: *"ผมจัดให้เพื่อนตลอด ผมไม่ได้อยากเป็นเหรัญญิก"*

### เขาคือใคร (จริง 100%)
- อายุ 52 (range 45-65), ชาย 90%
- เจ้าของ SME (พนง. 30-100) / ผู้บริหารระดับสูง
- A/B+ — กอล์ฟ ฿1,500-7,500/รอบ × 2-4 รอบ/เดือน
- Bangkok metro / ปริมณฑล (สนาม 50+ ในรัศมี 1 ชม.) · ทริปไกล Pattaya/Hua Hin/Chiang Mai ปีละ 2-4
- iPhone หลัก, บางคนมี Apple Watch/Garmin
- ไทย 100%, ไม่อยากอ่าน UI อังกฤษ

### ไลฟ์สไตล์
- **กอล์ฟ**: เล่นมา 10-20 ปี, handicap 15-25, ก๊วน 4-8 คน, ออกรอบเสาร์-อาทิตย์เช้า, รอบนึง 5-6 ชม.
- **ดิจิทัล**: LINE = ศูนย์กลางทุกอย่าง · PromptPay = จ่ายเพื่อน · บัตรเครดิตระวัง scam · FB หนัก, IG/TikTok น้อย · ไม่ค่อยโหลด app ใหม่
- **จัดก๊วน** [สมมติฐาน]: นัดใน LINE, คิดเงินบนกระดาษ/Excel, นั่งคิดที่ clubhouse 10-30 นาที, มี "ลูกหนี้รอบที่แล้ว" ค้างทุกครั้ง

### ความต้องการ
- **Functional**: จบรอบเร็ว, คิดถูก, มีหลักฐาน, ทุกคนจ่ายจริง
- **Emotional [สมมติฐาน]**: ไม่อยากเป็นจำเลย, อยากกลับมาสนุก, อยากดู "จัดการเป็น", ไม่อยากให้ก๊วนทะเลาะ

### ความกังวล (objections)
- product: "กระดาษก็คิดได้ ทำไมต้องจ่าย" (#1), "เพื่อนจะใช้ไหม", "กรอกบัญชีไหม", "เสียจังหวะรอบไหม"
- brand: "KuanGolf คือใคร", "จะหายไปไหม", "ระบบเงินผิดไหม"
- subscription: "auto-renew หลอกไหม", "cancel ยากไหม"

### เหตุผลที่ซื้อ [สมมติฐาน]
- **Triggers**: ทะเลาะครั้งล่าสุด, เพื่อน Captain ใช้, อายุเริ่มล้า, Founding Captain (scarcity)
- **Rational**: ฿999/ปี ≈ greenfee 1 หลุม, PromptPay ไม่ต้องใส่บัตร, LINE-native ไม่ลากเพื่อนโหลด, 7-day refund + no auto-renew

### สิ่งที่ทำให้ลังเล

| Objection | วิธี defuse |
|---|---|
| "กระดาษก็พอ" | แสดงเวลาประหยัด + หลักฐาน LINE |
| "เพื่อนจะใช้ไหม" | Free tier ให้เพื่อน + Group plan |
| "ระบบเงิน buggy ไหม" | QR ตรงคน-ต่อ-คน, ไม่ถือเงิน |
| "auto-renew หลอกไหม" | "ไม่มี auto-renew" ตัวใหญ่ |
| "KuanGolf คือใคร" | Bob เป็นหน้า + Thunder credibility |
| "รกในรอบไหม" | 1-click + แคดดี้ใช้ได้ + offline |

### ช่องทางที่เจอแบรนด์ [สมมติฐาน]
- **High**: เพื่อน Captain ใน LINE ก๊วน, FB ads (45-65 ชาย Bangkok), LINE OA broadcast, ที่สนาม (phase 2)
- **Medium**: Bob's network (COO/SME), Thunder ecosystem cross-promote, YouTube
- **Low (อย่าเสียเงิน)**: TikTok, IG, Google Search
- **อย่าเข้า**: golfdiggTODAY (= BirdieX media), TGA channel (ถ้ายังไม่ partnership)

## 5) Pain Point

> Pain ลึกสุด = **identity tension** ระหว่าง "ผู้บริหารที่ทุกคนเคารพ" กับ "เหรัญญิกก๊วนที่นั่งคิดเลขที่ clubhouse"

### ปัญหาที่ลูกค้าพูดออกมา (Surface — 3/10)
"คิดเลขนานทุกครั้ง", "เพื่อนจ่ายช้า ต้องทวง", "ลืมว่ารอบที่แล้วใครได้ใครเสีย", "นั่ง clubhouse 30 นาที" — symptom ไม่ใช่ pain จริง

### ความกลัวที่ซ่อนอยู่ (Hidden — pain จริง)
- 🔴 **identity**: กลัวกลายเป็น "พี่เอ" (archetype ที่ไม่อยากเป็น), กลัวดู "ตามไม่ทัน", กลัวเสีย authority, กลัวภาพ "เกี่ยงเรื่องเงิน"
- 🔴 **ความสัมพันธ์**: กลัวเสียเพื่อน 10-20 ปีเพราะเงิน 500-2,000, กลัวเป็นคน "ทวงเงิน", กลัวเพื่อนคิดว่าเอาเปรียบ
- 🔴 **practical**: กลัวบัตร scam, กลัว auto-renew หลอก, กลัวข้อมูลเงินรั่ว

### ความลังเลก่อนซื้อ (เรียงน้ำหนัก)
1. 🔴🔴🔴 Network effect — "ก๊วนผมไม่ใช้ tech"
2. 🔴🔴🔴 Trust แบรนด์ — "KuanGolf คือใคร"
3. 🔴🔴 Auto-renew scam fear
4. 🔴🔴 Performance ในรอบจริง — "ถ้าพัง = อายเพื่อน"
5. 🔴 Value vs Free / Time to learn / Privacy

### สิ่งที่เขาอยากได้จริง (True Want)
1. **Identity upgrade** — เป็น Captain เวอร์ชั่นที่ดีกว่า ไม่ใช่เลิกเป็น Captain
2. **Authority retention** — เป็นคนนำ ไม่ใช่คนรับใช้
3. **Friendship insurance** — ก๊วนอยู่ต่ออีก 10 ปี
4. **Joy reclamation** — กลับมาเล่นเพื่อกอล์ฟ
5. **Modern signaling** — ให้คนเห็นว่า "พี่เอก up-to-date"

### 3 ประโยคในใจลูกค้า
1. *"ผมเป็นเจ้าของบริษัท พนักงาน 50 คน — แต่ในก๊วน ผมเป็นพี่เอที่นั่งคำนวณ"* (identity tension)

## E2) Unit Economics + Financial สมมติฐาน

> 🔴 **คำเตือนหนัก**: ทุกตัวเลขด้านล่างเป็น [สมมติฐาน] ล้วน — pre-traction ไม่มี data จริงสักตัว
> ใช้เป็น "กรอบคิด" เท่านั้น ห้ามเอาไปใส่ board deck โดยไม่เขียน "projection/assumption"

### ARPU สมมติฐาน 🔴

| Tier | ราคา | สัดส่วน paid mix (เดา) | contribution |
|---|---|---|---|
| Pro ฿999 | 999 | 50% | 500 |
| Group ฿1,290 | 1,290 | 35% | 452 |
| Lifetime ฿2,490 | 2,490 (÷3 ปี = 830/ปี) | 15% | 125 |
| **Blended ARPU/ปี** | | | **~฿1,077** |

### Funnel สมมติฐาน 🔴

```
ก๊วน 100 ก๊วน เห็นแอป
  ↓ 40% Captain ลองโหลด        → 40 Captain
  ↓ 50% ใช้จริง 1 รอบ           → 20 active Captain
  ↓ 30% Free→Paid               → 6 paid Captain
  + เพื่อนในก๊วน 6 × 4-12 คน    → 24-72 active golfer (ฟรี/seed)
```

→ ต้อง validate ทุก % conversion ใน pilot เดือน 2-3

### CAC สมมติฐาน 🔴

| Channel | CAC ต่อ paid Captain (เดา) | หมายเหตุ |
|---|---|---|
| Captain referral | ฿0-100 | organic, ต่ำสุด — เป้าหลัก |
| FB ads | ฿300-800 | ต้องทดสอบจริง |
| LINE OA broadcast | ฿50-200 | ถ้ามีฐาน add แล้ว |
| ที่สนาม | ฿200-500 | + ค่า booth/signage |

### LTV สมมติฐาน 🔴

- ARPU ฿1,077/ปี × retention 🔴 (เดา 2-3 ปี เพราะ network lock-in) = **LTV ฿2,150-3,230**
- LTV:CAC ที่ดี ควร > 3:1 → CAC ต้องคุมไม่เกิน ~฿700-1,000
- **ถ้า referral เป็นช่องหลัก** → LTV:CAC อาจ > 10:1 (จุดแข็งของ network model)

### Break-even สมมติฐาน 🔴

- ต้องรู้ fixed cost/เดือนก่อน (รอ Bob: dev + marketing + CSM)
- ถ้า fixed ~฿50K/เดือน → ต้องมี ~46 paid Captain ใหม่/เดือน (ที่ ARPU ฿1,077) เพื่อ cover
- **ยังคำนวณจริงไม่ได้** จนกว่าจะรู้ cost structure

### ตัวเลขที่ต้องเก็บตั้งแต่วันแรก (instrumentation) 🟢
- Captain signup → activation (ใช้จริง 1 รอบ)
- Free → Paid conversion (per tier)
- รอบ/ก๊วน/เดือน (engagement)
- Referral coefficient (1 Captain ชวนได้กี่ก๊วน) — **ตัวชี้วัดสำคัญสุด**
- Churn rate (ปีที่ 2)

## E3) Moat Analysis + SWOT เต็ม

### Moat — เรียงตามความแข็งแรง

**1. Network effect ก๊วน** 🟢 (แข็งสุด)
- เพื่อน 1 คนใช้ → ก๊วน 4-12 คนใช้ตาม
- เปลี่ยน app ต้องเปลี่ยนพร้อมกันทั้งก๊วน = switching cost สูงมาก
- ยิ่งก๊วนใช้นาน ยิ่งมีประวัติ/ลูกหนี้ค้าง → ออกยาก

**2. Thunder Solution infra** 🟢 (แข็ง — เลียนแบบยาก)
- EasySlip + BoostSMS + EasyCRM + PromptPay = ทำ settlement flow ได้เร็ว/ถูก
- คู่แข่งต้องสร้าง infra นี้เองทั้งหมด = เสียเวลา 6-12 เดือน

**3. Cultural/language fit** 🟡 (กลาง — เลียนแบบได้แต่ช้า)
- ต่อแต้ม, halve-on-tie, System 36, ภาษาก๊วนไทย
- global player ต้อง localize ลึก (BirdieX ทำได้เพราะเป็นไทย)

**4. PromptPay settlement** 🟡 (กลาง — feature เลียนแบบได้)
- ตอนนี้ 0 competitor ทำ แต่เป็น feature → ลอกได้ใน 6-12 เดือน
- ต้องรีบใช้ช่วงที่ยังไม่มีใครทำ เพื่อสร้าง brand association

**5. Brand = "ตัวกลางเป็นกลาง"** 🟡 (อ่อน-กลาง — ต้องสร้างเวลา)
- ถ้า own คำว่า "ไม่มีใครเป็นจำเลย" ได้ก่อน = mental moat
- แต่ต้องลงทุน content/time

### SWOT เต็ม

**Strengths** 🟢
- Thunder infra (PromptPay/LINE/CRM พร้อม)
- Bob = COO (business sense + network)
- Blank slate (0 legacy constraint)
- Specialist focus (lean, ไม่แตก)
- Native Thai stack

**Weaknesses** 🟢
- Pre-traction (0 user/paying/data)
- 0 validated insight
- ไม่มี media engine (vs golfdigg/InVent)
- ไม่มี course distribution (vs SmartScore/BirdieX)
- มาทีหลัง 1-3 ปี
- ทีม/งบ marketing ยังไม่ชัด

**Opportunities** 🟡
- Layer 3 (settlement) ยังว่าง 0 competitor
- Window 12 เดือน ก่อนยักษ์ขยับ
- Captain network multiplier
- B2B สนาม (phase 2)
- Inbound golfer (phase 3)
- TGA partnership (ถ้าได้)

**Threats** 🟡
- 🔴 BirdieX/InVent push หนัก (ทุน + media + 150 สนาม)
- 🔴 Beezer ขยาย SEA + เพิ่ม PromptPay
- 🔴 18Birdies localize ไทย
- 🟡 TGA ออก scorecard app เอง
- 🟡 โลกัณฑ์กอล์ฟ เป็น direct match (ต้อง audit)
- 🟡 Apple Watch / Garmin native golf
- 🟡 Legal risk (PromptPay settlement = e-money/gambling?)

### TOWS — กลยุทธ์จับคู่ 🟡

| | Opportunities | Threats |
|---|---|---|
| **Strengths** | ใช้ Thunder infra เร่ง PromptPay settlement ครอง Layer 3 ก่อนใคร (SO) | ใช้ network effect สร้าง switching cost ก่อน Beezer/BirdieX มาถึง (ST) |
| **Weaknesses** | ใช้ Captain referral แก้ปัญหาไม่มี media engine (WO) | เร่ง validate + pilot ก่อน window 12 เดือนปิด มิฉะนั้นยักษ์กิน niche (WT) |

## E11) Competitive Defense ทุกคู่แข่ง

> สำหรับแต่ละคู่แข่ง: เขาจะโจมตียังไง + เราตอบยังไง + เมื่อไรต้องระวัง

### คู่แข่งที่ 1 — กระดาษ + LINE chat (enemy #1) 🟢
- **เขาชนะเรา**: ฟรี, คุ้นมือ, ไม่ต้องเรียนรู้
- **เราชนะเขา**: ลบ Captain blame + มีหลักฐาน + ส่ง QR ได้ + ไม่ค้างเงิน
- **มุมสื่อสาร**: ไม่โจมตีกระดาษตรงๆ — บอกว่า "กระดาษทำให้คุณเป็นจำเลย"
- **ระวังเมื่อ**: ลูกค้าไม่เห็น pain ของ blame → ต้อง educate ด้วย story

### คู่แข่งที่ 2 — โลกัณฑ์กอล์ฟ (direct match ที่สุด) 🟡
- **เขาชนะเรา**: ไทย, มีอยู่แล้ว, calculator เดิมพันก๊วน 9/18/27/36
- **เราชนะเขา**: + LINE-native + PromptPay QR + network ก๊วน + UX ใหม่
- **มุมสื่อสาร**: ไม่เอ่ยชื่อ — เน้น "จบแล้วส่ง QR เข้า LINE เลย" (โลกัณฑ์ฯ ไม่มี)
- **ระวังเมื่อ**: ต้อง **audit จริงก่อน** (ยังไม่ได้ลอง) — ถ้าเขาเพิ่ม LINE/PromptPay = อันตราย
- **🚨 Action**: install ลองใน 5 วัน

### คู่แข่งที่ 3 — Beezer Golf (feature ใกล้สุด global) 🟡
- **เขาชนะเรา**: 28+ games, BEEZER Bank (ledger ข้ามรอบ), 40K courses
- **เราชนะเขา**: ไทย + LINE + PromptPay (Beezer ไม่มีทั้งหมด)
- **มุมสื่อสาร**: ไม่เทียบตรง — own "PromptPay + LINE" ที่เขาทำไม่ได้
- **ระวังเมื่อ**: 🔴 Beezer ขยาย SEA + เพิ่ม PromptPay → moat หาย
- **🚨 Action**: install audit + เร่งสร้าง network switching cost

### คู่แข่งที่ 4 — BirdieX/golfdigg (ทุนหนาสุดในไทย) 🟡
- **เขาชนะเรา**: InVent backing, 150+ สนาม, media engine, social, beta แล้ว
- **เราชนะเขา**: specialist settlement (BirdieX = generalist social+booking)
- **มุมสื่อสาร**: ไม่ชนตรง — เขาจับ Match ใหญ่/social, เราจับ settlement ก๊วนเล็ก
- **ระวังเมื่อ**: 🔴 InVent push หนัก Q3-Q4 → window แคบลง
- **กลยุทธ์**: grab Captain niche ก่อน 12 เดือน + อย่าเล่นเกม social

### คู่แข่งที่ 5 — 18Birdies/Hole19 (global leader) 🟢
- **เขาชนะเรา**: feature ครบ, brand, GPS/AI, 10M users
- **เราชนะเขา**: ไทย + Captain pain + LINE + PromptPay + ราคา
- **มุมสื่อสาร**: ไม่แตะ GPS/AI — own "ก๊วนไทย" ที่เขาไม่เข้าใจ
- **ระวังเมื่อ**: 🟡 ออกภาษาไทย → เร่ง localization + cultural moat

### คู่แข่งที่ 6 — SmartScore (B2B threat) 🟢
- **เขาชนะเรา**: course infra, ทุน, B2B network, เจ้าของสนาม
- **เราชนะเขา**: consumer-first, ไม่ผูกสนาม, Captain-focused
- **มุมสื่อสาร**: คนละเกม — เขา B2B สนาม, เรา B2C Captain
- **ระวังเมื่อ**: phase 2 ตอนเราเข้า B2B → เลี่ยงชนสนาม Artitaya

### สรุปกลยุทธ์ป้องกัน 🟢
1. **อย่าชนใครตรงๆ** — own Layer 3 (settlement) ที่ทุกคนมองข้าม
2. **เร่ง network effect** — switching cost คือ moat เดียวที่ทุนหนากว่าก็แซงยาก
3. **PromptPay + LINE** = ป้อมที่ global ลอกช้า, ไทยต้องสร้าง infra
4. **Window 12 เดือน** — Beezer/BirdieX คือนาฬิกาจับเวลา

## E12) Brand Voice เต็ม + ตัวอย่าง

### หลักการเสียง (Voice Principles)

**1. พูดเหมือนเพื่อน Captain อาวุโส** 🟢
- เข้าใจชีวิตก๊วน, ไม่ต้องอธิบายว่า "ต่อแต้ม" คืออะไร
- ตรงไปตรงมา ไม่อ้อมค้อม

**2. ภาษาคน 100%** 🟢
- ไม่ใช้ศัพท์ tech/agency
- "เริ่มใช้" ไม่ใช่ "onboard", "หน้าหลัก" ไม่ใช่ "dashboard"

**3. มั่นใจ ไม่ขายดี** 🟢
- ของจริงขายตัวเองได้ ไม่ต้อง hyperbole
- ไม่พูด "ปฏิวัติวงการ/เปลี่ยนโลก"

**4. เคารพ ไม่ดูถูก** 🟢
- ไม่ assume ว่าลูกค้า tech-savvy
- แต่ไม่ทำเหมือนเขาโง่

### Do / Don't ตัวอย่างจริง

| สถานการณ์ | ✅ Do | ❌ Don't |
|---|---|---|
| Headline | "หัวก๊วน ไม่ต้องเป็นเหรัญญิก" | "Revolutionary Golf Scoring Platform" |
| อธิบาย feature | "กดสกอร์ แอปคิดให้ ส่ง QR เข้า LINE" | "AI-powered auto-calculation engine" |
| CTA | "ลองฟรี ไม่ต้องใส่บัตร" | "Start your free trial now!" |
| ราคา | "ปีนึง เท่า greenfee หลุมเดียว" | "Only $30/year subscription" |
| Error/ปัญหา | "เน็ตหลุดไม่เป็นไร เดี๋ยวเซฟให้" | "Connection error 503" |
| ทักทาย LINE OA | "ยินดีต้อนรับหัวก๊วน 👋" | "Welcome to KuanGolf onboarding" |

### Tagline library (ใช้ได้ตามบริบท) 🟡

**หลัก**:
- หัวก๊วน ไม่ต้องเป็นเหรัญญิก
- ก๊วนไทย เพื่อหัวก๊วน

**ตามแคมเปญ**:
- จบรอบ จบเรื่องเงิน (เน้น speed)
- ไม่มีใครเป็นจำเลย (เน้น neutral)
- อย่าเป็นพี่เอ เป็น Captain ยุคใหม่ (เน้น identity, ใช้ anti-persona)
- ก๊วนสนุก จบในรอบ ส่งเข้า LINE (เน้น workflow)

### Channel-specific tone 🟡

| Channel | ปรับโทนยังไง |
|---|---|
| **FB ads** | สั้น สะดุด ใช้ภาพ before/after, hook ด้วย pain |
| **LINE OA** | เป็นกันเอง เหมือนเพื่อนทัก, emoji ได้นิดหน่อย |
| **Landing page** | มั่นใจ มีหลักฐาน เน้น trust signal |
| **Sales (1:1)** | ฟังก่อน ถาม "ทะเลาะครั้งล่าสุดเมื่อไร" |
| **Investor** | English OK, เน้น market + wedge + moat |

### คำต้องห้าม (ย้ำจาก Part I) 🟢
- legal: wallet, บัญชี, เงินในแอป, เดิมพัน, พนัน, betting, bank
- เหมือนคู่แข่ง: all-in-one, GPS, handicap tracker, social golf, AI-powered
- โทนผิด: ปฏิวัติวงการ, เปลี่ยนโลก, ทะลึ่ง, self-deprecating
