# 📚 แหล่งข้อมูลและความเชื่อมั่น

> กลับไป [[00 แผนที่ความรู้ (MOC)]]

## ระดับความเชื่อมั่นของข้อมูล
- **ฝั่ง Facebook/Meta** ([[05 ช่องทาง Facebook และนโยบาย Meta]]) — จาก **เอกสารทางการ Meta ล้วน + ตรวจสอบไขว้ 3 เสียง** → เชื่อถือได้สูง
- **EasySlip/คู่แข่ง/บริษัท** ([[06 EasySlip และ Thunder API]], [[07 คู่แข่งในตลาด]], [[10 ข้อมูลบริษัท Thunder Group]]) — จาก **เว็บ/เอกสารทางการของผู้ให้บริการเอง (first-party)** ยังไม่ได้ตรวจสอบไขว้เชิงลึก → **ยืนยันตัวเลขซ้ำก่อนตัดสินใจลงทุน**
- ⚠️ **นโยบาย Meta + ราคา อ้างอิง ณ ก.ค. 2026** — Meta เปลี่ยนกฎบ่อย verify ซ้ำก่อนลงมือจริง

## ที่มา (หลัก)
- **Meta for Developers:** Messenger Platform (webhooks, messages, policy, permissions, utility messages, changelog), Instagram Platform webhooks, Graph API rate limiting, Platform Terms
- **document.easyslip.com** — API v2 reference, error codes, slip-age-support, authentication
- **easyslip.com, thunder.in.th** — ราคา/ช่องทาง/บทความ
- **creden.co / dataforthai** — ข้อมูลจดทะเบียน DBD
- **ITMX** (Standard API Transaction Verification), **ธปท./MDES** (พ.ร.ก.ไซเบอร์ ฉบับ 2), ข่าวสถิติสลิปปลอม
- **คู่แข่ง:** zwiz.ai, chatcone.com, page365.net, commerzy.com, slipok.com, slip.rdcw.co.th, apiportal.kasikornbank.com, developer.scb
- ไฟล์ `Thunder_Solution_Public_Context_AI-safe.md` (Bobsanchez, 2026-07-09)

## Methodology
งานวิจัยผ่าน deep-research 2 สาย รวม **212 agent** · ดึง **46 แหล่ง** · ตรวจสอบไขว้ **50 claim** (ยืนยัน 46, หักล้างทิ้ง 4) · ฝั่ง Facebook/Meta ยืนยันจากเอกสารทางการ Meta ล้วน
