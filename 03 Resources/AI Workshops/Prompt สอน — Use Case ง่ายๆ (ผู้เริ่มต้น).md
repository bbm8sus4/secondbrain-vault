# Prompt สอน — Use Case ง่ายๆ (สำหรับผู้เริ่มต้น)

คลัง prompt พร้อมใช้ สำหรับสอนคนที่เพิ่งเริ่มใช้ AI (Claude / Claude Code) — **ก็อปทั้งก้อนไปวางได้เลย** แล้วแก้ส่วนใน `[...]` ให้เป็นของตัวเอง

**วิธีใช้ในคลาส:**
- Prompt ที่ **สร้างแอป HTML** → ใช้กับ Claude Code (สร้างไฟล์ให้) หรือ Claude.ai (โชว์ใน Artifact)
- Prompt งานเอกสาร/การตลาด/วิเคราะห์ → ใช้ที่ไหนก็ได้ (Claude.ai / Claude Code / มือถือ)

---

## โครงสร้าง Prompt ที่ดี (สอนก่อนลงมือ)

Prompt ดี = บอกให้ครบ 4 อย่าง จำง่ายๆ ว่า **"ใคร-ทำอะไร-แบบไหน-ห้ามอะไร"**

| ส่วน | คือ | ตัวอย่าง |
|---|---|---|
| **บริบท/บทบาท** | ให้ AI สวมบทหรือรู้สถานการณ์ | "คุณเป็นนักบัญชี ช่วย…" |
| **งานที่ต้องการ** | บอกชัดว่าจะเอาอะไร | "สร้างเว็บแอปบัญชีรายรับรายจ่าย" |
| **รูปแบบผลลัพธ์** | อยากได้ออกมาหน้าตาไง | "เป็นไฟล์ HTML เดียว, ตาราง, ภาษาไทย" |
| **ข้อจำกัด/เงื่อนไข** | กติกาที่ต้องทำตาม | "ไม่ต้องมี server, ไม่ใช้ library นอก" |

> ยิ่งบอกละเอียด ยิ่งได้ตรงใจ — แต่ไม่ต้องกลัวยาว AI ชอบรายละเอียด

## หมวด 1 — สร้างแอป/เครื่องมือ HTML (เห็นผลจับต้องได้)

### 1.1 ระบบบัญชีรายรับ-รายจ่าย (ตัวอย่างหลัก)
```
สร้างเว็บแอปบัญชีรายรับ-รายจ่ายเป็นไฟล์ HTML ไฟล์เดียว ใช้งานได้จริงบนมือถือและคอม มีฟีเจอร์:
- ฟอร์มเพิ่มรายการ: วันที่, ประเภท (รายรับ/รายจ่าย), หมวดหมู่ (อาหาร/เดินทาง/เงินเดือน/ช้อปปิ้ง/อื่นๆ), จำนวนเงิน, โน้ต
- ตารางรายการทั้งหมด เรียงจากล่าสุด แก้ไข/ลบได้
- สรุปยอด: รายรับรวม, รายจ่ายรวม, คงเหลือ (ของเดือนที่เลือก)
- กราฟวงกลมแยกรายจ่ายตามหมวดหมู่
- ตัวกรองตามเดือน
- บันทึกข้อมูลใน localStorage (ปิดแล้วเปิดใหม่ข้อมูลยังอยู่ ไม่ต้องมี server)
- ปุ่ม export เป็นไฟล์ CSV
ดีไซน์: สะอาด โทนสว่าง อ่านง่าย ภาษาไทย ตัวเลขใส่คอมมา สกุลเงินบาท
ข้อจำกัด: เขียนเป็น vanilla HTML/CSS/JS ในไฟล์เดียว ไม่ใช้ library ภายนอก
```
**ปรับ:** เปลี่ยนหมวดหมู่ / เพิ่มงบประมาณต่อเดือน / เพิ่มกราฟแนวโน้มรายเดือน

### 1.2 To-Do List / เช็คลิสต์งาน
```
สร้าง To-Do List เป็นไฟล์ HTML เดียว: เพิ่ม/ลบ/ติ๊กเสร็จงานได้, แยกหมวด (วันนี้/สัปดาห์นี้/รอไว้), ลากจัดลำดับได้, บันทึกใน localStorage, โชว์จำนวนงานที่เหลือ ดีไซน์สะอาดโทนสว่าง ภาษาไทย ไม่ใช้ library ภายนอก
```

### 1.3 ตัวติดตามนิสัย (Habit Tracker)
```
สร้าง Habit Tracker เป็นไฟล์ HTML เดียว: เพิ่มนิสัยที่อยากทำ (เช่น ออกกำลังกาย/ดื่มน้ำ), ติ๊กเช็คแต่ละวันเป็นปฏิทินรายเดือน, แสดง streak (ทำติดต่อกันกี่วัน) และ % สำเร็จ, บันทึกใน localStorage ดีไซน์สว่าง ภาษาไทย ไฟล์เดียว ไม่ใช้ library นอก
```

### 1.4 ใบเสนอราคา / ใบแจ้งหนี้
```
สร้างเครื่องมือออกใบเสนอราคาเป็นไฟล์ HTML เดียว: กรอกชื่อบริษัท/ลูกค้า, เพิ่มรายการสินค้า (ชื่อ/จำนวน/ราคาต่อหน่วย) คิดยอดรวม + VAT 7% อัตโนมัติ, มีเลขที่เอกสารและวันที่, ปุ่มพิมพ์เป็น PDF (window.print แบบจัดหน้าสวย) ภาษาไทย สกุลเงินบาท ไฟล์เดียว ไม่ใช้ library นอก
```

### 1.5 เครื่องคิดเลขสุขภาพ (BMI + แคลอรี่)
```
สร้างเครื่องคำนวณสุขภาพเป็นไฟล์ HTML เดียว: กรอกเพศ/อายุ/ส่วนสูง/น้ำหนัก/ระดับกิจกรรม แล้วคำนวณ BMI (พร้อมบอกเกณฑ์), BMR และแคลอรี่ที่ควรกินต่อวัน, แนะนำน้ำหนักเป้าหมาย ดีไซน์สว่าง ภาษาไทย ไฟล์เดียว ไม่ใช้ library นอก
```

### 1.6 ตัวติดตามเป้าหมายเงินออม
```
สร้างแอปติดตามเป้าหมายเงินออมเป็นไฟล์ HTML เดียว: ตั้งเป้าหมาย (ชื่อ/จำนวนเงิน/วันครบกำหนด), บันทึกยอดที่ออมแต่ละครั้ง, โชว์ progress bar %, บอกว่าต้องออมเดือนละเท่าไหร่ถึงจะทัน, บันทึกใน localStorage ภาษาไทย บาท ไฟล์เดียว ไม่ใช้ library นอก
```

## หมวด 2 — งานเอกสาร / ออฟฟิศ (ใช้ทุกวัน)

### 2.1 สรุปประชุมจากโน้ตดิบ
```
นี่คือโน้ตประชุมแบบดิบๆ ช่วยสรุปให้เป็น: (1) หัวข้อที่คุยกัน (2) ข้อสรุป/มติ (3) สิ่งที่ต้องทำต่อ (ใคร-ทำอะไร-ภายในเมื่อไหร่) เขียนกระชับ อ่านง่าย ภาษาไทย
[วางโน้ตประชุมตรงนี้]
```

### 2.2 ร่างอีเมลทางการ
```
ช่วยร่างอีเมลภาษาไทยแบบสุภาพเป็นทางการ ถึง [ลูกค้า/พาร์ทเนอร์] เรื่อง [เรื่องที่จะสื่อ] โดยมีประเด็น: [ใส่ประเด็น 2-3 ข้อ] โทนสุภาพ มืออาชีพ กระชับ ไม่เยิ่นเย้อ ปิดท้ายด้วยการขอบคุณ
```

### 2.3 แปลไทย↔อังกฤษ แบบมืออาชีพ
```
ช่วยแปลข้อความนี้เป็น [ภาษาอังกฤษ] ให้เป็นธรรมชาติแบบเจ้าของภาษา ไม่แปลตรงตัว รักษาโทนให้เหมาะกับ [อีเมลธุรกิจ/โพสต์การตลาด/แชทเพื่อน] ถ้ามีคำที่แปลได้หลายแบบ ให้เลือกที่เหมาะที่สุดแล้วบอกเหตุผลสั้นๆ
[วางข้อความ]
```

### 2.4 สรุปเอกสาร/บทความยาว
```
ช่วยสรุปเอกสารนี้ให้เหลือใจความสำคัญ: เริ่มด้วยสรุป 3 บรรทัด แล้วตามด้วย bullet ประเด็นหลัก และปิดท้ายด้วย "สิ่งที่ควรทำต่อ" ภาษาไทย เข้าใจง่าย
[วางเนื้อหา หรือ ลากไฟล์ใส่]
```

### 2.5 ทำ SOP / คู่มือขั้นตอนงาน
```
ช่วยแปลงขั้นตอนงานนี้ให้เป็นคู่มือ SOP ที่พนักงานใหม่อ่านแล้วทำตามได้เลย: มีหัวข้อ, ขั้นตอนเป็นข้อๆ ชัดเจน, จุดที่ต้องระวัง, และเช็คลิสต์ท้ายสุด ภาษาไทย
[อธิบายงานคร่าวๆ]
```

## หมวด 3 — การตลาด / คอนเทนต์

### 3.1 ไอเดียคอนเทนต์ 30 วัน
```
ช่วยคิดไอเดียคอนเทนต์ 30 หัวข้อ สำหรับ [ธุรกิจ/สินค้า] กลุ่มเป้าหมาย [ใคร] ช่องทาง [Facebook/TikTok/LINE] จัดเป็นตาราง: วันที่ | ประเภท (ให้ความรู้/ขาย/สร้างความสัมพันธ์) | หัวข้อ | ไอเดียคร่าวๆ ให้หลากหลาย ไม่ซ้ำแนว ภาษาไทย
```

### 3.2 เขียนแคปชั่น + แฮชแท็ก
```
ช่วยเขียนแคปชั่นโพสต์ [Facebook/IG] ขาย [สินค้า] จุดขายคือ [...] กลุ่มเป้าหมาย [...] เขียนมา 3 เวอร์ชัน (โทนสนุก / โทนน่าเชื่อถือ / โทนกระตุ้นให้รีบซื้อ) พร้อมแฮชแท็กที่เกี่ยวข้อง ภาษาไทย
```

### 3.3 Ad Copy หลายเวอร์ชัน (A/B Test)
```
ช่วยเขียนข้อความโฆษณาสั้นๆ สำหรับ [สินค้า/บริการ] มา 5 เวอร์ชัน เน้นมุมต่างกัน (ราคา / ประหยัดเวลา / แก้ปัญหา / รีวิว / ความเร่งด่วน) แต่ละอันไม่เกิน 2 บรรทัด มี hook ที่หยุดนิ้วได้ ภาษาไทย
```

### 3.4 สร้าง FAQ จากข้อมูลสินค้า
```
จากข้อมูลสินค้านี้ ช่วยสร้างชุดคำถาม-คำตอบ (FAQ) 10 ข้อ ที่ลูกค้ามักถามก่อนซื้อ ตอบให้ชัดเจน สั้น น่าเชื่อถือ ภาษาไทย
[วางรายละเอียดสินค้า]
```

## หมวด 4 — ข้อมูล / วิเคราะห์

### 4.1 วิเคราะห์ไฟล์ Excel/CSV
```
นี่คือไฟล์ข้อมูล [ยอดขาย/ลูกค้า/สินค้า] ช่วยวิเคราะห์ให้: (1) ภาพรวมมีอะไรน่าสนใจ (2) 3 อันดับสูงสุด/ต่ำสุด (3) แนวโน้มที่เห็น (4) ข้อเสนอแนะที่ทำได้จริง สรุปเป็นภาษาไทยเข้าใจง่าย ไม่ต้องเทคนิคเยอะ
[ลากไฟล์ใส่ หรือวางข้อมูล]
```

### 4.2 สร้าง Dashboard จากข้อมูล
```
จากข้อมูลนี้ ช่วยสร้าง dashboard เป็นไฟล์ HTML เดียว: มีการ์ดตัวเลขสรุปด้านบน (ยอดรวม/เฉลี่ย/จำนวน), กราฟแท่งและกราฟเส้นแสดงแนวโน้ม, ตารางข้อมูล ดีไซน์สะอาดโทนสว่าง ภาษาไทย ตัวเลขมีคอมมา ไม่ใช้ library ภายนอก (วาดกราฟด้วย canvas/SVG เอง)
[วางข้อมูล]
```

### 4.3 เปลี่ยนตัวเลขเป็นกราф
```
ช่วยเปลี่ยนตัวเลขชุดนี้เป็นกราฟที่อ่านง่าย เลือกชนิดกราฟที่เหมาะสมที่สุดแล้วบอกเหตุผล ทำเป็นไฟล์ HTML เดียว ภาษาไทย
[วางตัวเลข]
```

## หมวด 5 — เรียนรู้ / ค้นคว้า

### 5.1 อธิบายเรื่องยากให้เข้าใจง่าย
```
ช่วยอธิบาย [เรื่องที่อยากเข้าใจ] แบบที่คนไม่มีพื้นฐานก็เข้าใจได้ ใช้การเปรียบเทียบกับของใกล้ตัว ยกตัวอย่างจริง 1 อัน และสรุปใน 1 ประโยคว่าทำไมเรื่องนี้สำคัญ ภาษาไทย
```

### 5.2 สร้างแบบทดสอบ (Quiz)
```
ช่วยสร้างแบบทดสอบ 10 ข้อ เรื่อง [หัวข้อ] แบบปรนัย 4 ตัวเลือก ไล่จากง่ายไปยาก พร้อมเฉลยและคำอธิบายสั้นๆ ท้ายข้อ ทำเป็นไฟล์ HTML ที่ทำข้อสอบแล้วเช็คคะแนนได้เลย ภาษาไทย
```

### 5.3 วางแผนเรียนรู้ทักษะใหม่
```
อยากเรียน [ทักษะ] จากศูนย์ ให้เก่งพอใช้งานจริงใน [ระยะเวลา] ช่วยวางแผนเป็นสัปดาห์: แต่ละสัปดาห์เรียนอะไร ฝึกอะไร มีแหล่งเรียนแนะนำ และวิธีวัดว่าเก่งขึ้น ภาษาไทย
```

## เทคนิคสอนเพิ่ม (บอกผู้เรียน)

- **ไม่ต้องเป๊ะครั้งเดียว** — ได้ผลมาแล้วสั่งต่อได้ เช่น "เปลี่ยนสีเป็นโทนเขียว" "เพิ่มปุ่มลบ" "ทำให้มือถือใช้ง่ายขึ้น"
- **ลากไฟล์ใส่ได้** — Excel, PDF, รูป วางเข้าไปแล้วสั่งงานกับมันได้เลย
- **สั่งเป็นภาษาไทยได้** — ไม่ต้องแปลเป็นอังกฤษก่อน
- **ขอหลายเวอร์ชัน** — "ทำมา 3 แบบให้เลือก" ดีกว่าขอทีละอัน
- **บอกกลุ่มเป้าหมาย** — "อธิบายให้เด็ก ป.6 เข้าใจ" กับ "อธิบายให้ CFO" ได้คนละคำตอบ
