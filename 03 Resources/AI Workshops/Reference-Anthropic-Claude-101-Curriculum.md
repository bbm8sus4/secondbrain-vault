## เทียบกับ workshop ขอนแก่นอิเล็คทริค

**คอร์สจริงของ Anthropic** เน้น 4 concept นี้เป็นแกน — ที่ workshop เรายังไม่ได้ครอบคลุมครบ:

| Concept | สำคัญยังไง | ทีมการตลาดใช้ทำอะไร |
|---|---|---|
| **Artifacts** | canvas/preview panel สร้าง doc/code/HTML แบบ live | ร่าง landing page · caption · brochure ให้เห็นภาพก่อน export |
| **Projects** | เก็บ context+file ถาวรต่อโปรเจกต์ ไม่ต้อง paste ซ้ำ | 1 project = 1 แบรนด์/แคมเปญ ใส่ CI + brief ครั้งเดียว |
| **Skills** | reusable prompt/behavior ที่ Claude เรียกใช้เอง | prompt library ที่เราตั้งใจจะสร้าง = คือ skills นั่นเอง |
| **Cowork / Code modes** | UI mode แยกสำหรับงาน chat vs สร้างชิ้นงาน | สลับ mode ให้เข้ากับจังหวะงาน |

## Top 5 prompt techniques (จาก Anthropic tutorial)

1. Use XML tags for structure
2. Ask for evidence before conclusions
3. Assign specific roles
4. Request step-by-step reasoning
5. Show examples instead of writing long instructions

## ข้อสังเกต

- คอร์ส 101 ของแท้เขา **ไม่สอน "prompt framework" (CRAFT/CRIT/RTF ฯลฯ)** โดยตรง — เขาใช้คำง่ายๆ ว่า "Getting better results" แล้วสอนผ่านตัวอย่าง
- **Artifacts** เป็น killer feature ที่ workshop เรายังไม่ได้พูดถึง — เหมาะกับทีมการตลาดมาก
- **Projects** เขาสอนเป็นบทเต็มในโมดูล 2 — ที่ workshop 7 Groups ใช้แนวคิดนี้แล้ว (ChatGPT Projects) แต่ขอนแก่นยังไม่มี
- คอร์สของเขาใช้เวลา 2-3 ชม. → workshop 1 วันของเราเทียบได้กับ 101 + Marketing use-case + hands-on deploy
