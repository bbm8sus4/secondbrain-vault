Setup Pack — ติดตั้ง Claude Code

Windows + macOS · กดปุ่ม คัดลอก ที่แต่ละบล็อกแล้วเอาไปวางได้เลย · คำสั่งเช็คกับ docs ทางการ 2026-07-11

!
ต้องมีก่อน: Claude Pro / Max / Team subscription — แพลนฟรีของ Claude.ai ใช้ Claude Code ไม่ได้ · macOS 13+ หรือ Windows 10 (1809+)

A1 ติดตั้ง Claude Code

วิธีแนะนำ — native installer (อัปเดตตัวเองอัตโนมัติ)

curl -fsSL https://claude.ai/install.sh | bash
— หรือ ถ้าใช้ Homebrew อยู่แล้ว —
brew install --cask claude-code

ยังไม่มี Homebrew? ติดตั้งก่อนด้วยบรรทัดนี้:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

A2 ติดตั้ง jq (จำเป็นถ้าจะใช้ status line)

brew install jq

A3 เริ่มใช้ + ล็อกอิน

พิมพ์แล้วมันจะเปิดเบราว์เซอร์ให้ล็อกอิน

claude

A4 ตรวจว่าติดตั้งสำเร็จ

claude --version
claude doctor
ใช้ PowerShell — Windows รองรับ native แล้ว ไม่ต้องพึ่ง WSL · แนะนำติดตั้ง Git for Windows ด้วย เพื่อเปิดใช้ Bash tool

B1 ติดตั้ง Claude Code — วางใน PowerShell

วิธีแนะนำ — native installer (อัปเดตตัวเองอัตโนมัติ)

irm https://claude.ai/install.ps1 | iex
— หรือ ผ่าน winget —
winget install Anthropic.ClaudeCode

B2 ติดตั้ง Git for Windows (แนะนำ)

winget install Git.Git

ถ้า Claude หา bash ไม่เจอ ให้เพิ่มบรรทัดนี้ใน ~/.claude/settings.json:

{ "env": { "CLAUDE_CODE_GIT_BASH_PATH": "C:\\Program Files\\Git\\bin\\bash.exe" } }

B3 ติดตั้ง jq (จำเป็นถ้าจะใช้ status line)

winget install jqlang.jq

B4 เริ่มใช้ + ล็อกอิน

พิมพ์แล้วมันจะเปิดเบราว์เซอร์ให้ล็อกอิน

claude

B5 ตรวจว่าติดตั้งสำเร็จ

claude --version
claude doctor

C ตั้งค่าง่ายสุด — ให้ Claude ทำให้เอง

หลังติดตั้ง+ล็อกอินเสร็จ เปิด claude ขึ้นมา แล้ววาง prompt นี้ในแชท มันจะดู OS แล้วตั้งค่าให้เหมาะเอง

ช่วยตั้งค่า Claude Code บนเครื่องนี้ให้พร้อมใช้งาน ตรวจ OS ก่อนแล้วทำให้เหมาะกับเครื่อง ห้ามแก้ไฟล์อื่นนอกจากที่จำเป็น:
1. สำรอง ~/.claude/settings.json ก่อน (ถ้ายังไม่มีให้สร้างใหม่)
2. ตั้งโมเดลเริ่มต้นเป็น Opus รุ่นล่าสุด (เพิ่ม key "model" ใน settings.json) — ถ้าฉันอยากได้รุ่นเจาะจงจะบอกทีหลัง
3. ตั้ง status line หลายแถว (โมเดล + context + usage + git branch) ให้รองรับ OS ปัจจุบัน และเช็คว่ามี jq แล้ว
4. ตรวจ claude doctor ว่าไม่มี error
5. สรุปว่าตั้งอะไรไปบ้าง ค่าเดิมคืออะไร และต้อง restart ไหม

D ตั้งค่าแบบ Manual

แก้ไฟล์ ~/.claude/settings.json (Windows: C:\Users\<ชื่อ>\.claude\settings.json) ใส่ค่านี้:

{
  "model": "claude-opus-4-6[1m]",
  "statusLine": { "type": "command", "command": "bash ~/.claude/statusline.sh", "padding": 0 }
}

อยากใช้โมเดล default → ลบบรรทัด "model" ออก · status line ต้องมีไฟล์สคริปต์ด้วย (installer อยู่ในคู่มือหมวด 10) · แก้แล้วต้อง restart

อยากล็อกโมเดลเป็นรุ่นเจาะจง (เช่น Opus 4.6) หรือดู status line ละเอียด → ดูคู่มือ Claude Code หมวด 9-10
