# 10 · เทมเพลต — ปิดปรับปรุงระบบ (Maintenance)

> Tier: **T2 ตามแผน** — ขออนุมัติ C ล่วงหน้า ≥ 3 วันทำการ
> ส่ง 4 จังหวะ: ล่วงหน้า 3 วัน → ย้ำ 1 วัน → ก่อนเริ่ม 1 ชม. (เฉพาะงานกระทบหนัก, ทาง SMS) → เสร็จแล้ว

## ตัวแปรที่ต้องเติม

| ตัวแปร | ตัวอย่าง |
|---|---|
| `{{แบรนด์}}` | EasyCRM |
| `{{ระบบที่กระทบ}}` | ระบบสะสมแต้มและตรวจสลิป |
| `{{วันที่}}` | วันจันทร์ที่ 30 มิ.ย. 2569 |
| `{{เวลาเริ่ม}}`–`{{เวลาสิ้นสุด}}` | 23:30–01:00 น. |
| `{{ผลกระทบ}}` | ไม่สามารถตรวจสลิปและสะสมแต้มได้ชั่วคราว |
| `{{ช่องทางติดต่อ}}` | LINE OA @easycrm |

## จังหวะ 1 — แจ้งล่วงหน้า 3 วัน

### LINE OA

```
[{{แบรนด์}}] แจ้งปิดปรับปรุงระบบล่วงหน้าค่ะ

{{วันที่}} เวลา {{เวลาเริ่ม}}–{{เวลาสิ้นสุด}}
ระบบที่กระทบ: {{ระบบที่กระทบ}}
ช่วงเวลาดังกล่าว {{ผลกระทบ}}

ทีมงานเลือกช่วงเวลาที่มีการใช้งานน้อยที่สุด เพื่อให้กระทบลูกค้าน้อยที่สุดค่ะ
ขออภัยในความไม่สะดวก สอบถามเพิ่มเติมได้ที่ {{ช่องทางติดต่อ}} ค่ะ
```

### Email — หัวเรื่อง: `[{{แบรนด์}}] แจ้งกำหนดการปิดปรับปรุงระบบ {{วันที่}}`

```
เรียนลูกค้าผู้ใช้บริการ {{แบรนด์}}

บริษัทขอแจ้งกำหนดการปิดปรับปรุงระบบ ดังนี้

วันเวลา: {{วันที่}} เวลา {{เวลาเริ่ม}}–{{เวลาสิ้นสุด}}
ระบบที่ได้รับผลกระทบ: {{ระบบที่กระทบ}}
ผลกระทบระหว่างดำเนินการ: {{ผลกระทบ}}

การปรับปรุงครั้งนี้มีวัตถุประสงค์เพื่อ {{เหตุผลสั้น ๆ เช่น เพิ่มความเสถียรของระบบ}}
หลังเวลา {{เวลาสิ้นสุด}} ระบบจะกลับมาใช้งานได้ตามปกติ หากพ้นกำหนดแล้วยังพบปัญหา
กรุณาติดต่อ {{ช่องทางติดต่อ}}

ขออภัยในความไม่สะดวก และขอบคุณที่ไว้วางใจใช้บริการค่ะ
ทีมงาน {{แบรนด์}}
```

## จังหวะ 2 — ย้ำ 1 วันก่อน

### LINE OA

```
[{{แบรนด์}}] เตือนอีกครั้งค่ะ พรุ่งนี้มีปิดปรับปรุงระบบ

{{วันที่}} เวลา {{เวลาเริ่ม}}–{{เวลาสิ้นสุด}}
ช่วงดังกล่าว {{ผลกระทบ}}

แนะนำให้ทำรายการที่จำเป็นก่อนเวลา {{เวลาเริ่ม}} นะคะ
ขออภัยในความไม่สะดวกค่ะ
```

---

## จังหวะ 3 — ก่อนเริ่ม 1 ชม. (SMS · เฉพาะงานที่กระทบหนัก)

> ≤ 70 ตัวอักษรไทย = 1 เครดิต — นับก่อนส่ง

```
{{แบรนด์}}: ปิดปรับปรุง {{เวลาเริ่ม}}-{{เวลาสิ้นสุด}}คืนนี้ งดใช้งานชั่วคราว
```

(ตัวอย่างจริง 58 ตัวอักษร: `EasyCRM: ปิดปรับปรุง 23:30-01:00 คืนนี้ งดตรวจสลิปชั่วคราว`)

---

## จังหวะ 4 — เสร็จแล้ว

### LINE OA

```
[{{แบรนด์}}] ปรับปรุงระบบเสร็จเรียบร้อยแล้วค่ะ

ระบบกลับมาใช้งานได้ตามปกติตั้งแต่เวลา {{เวลาที่เสร็จจริง}}
ขอบคุณที่รอ และขออภัยในความไม่สะดวกค่ะ

หากพบปัญหาการใช้งาน แจ้งได้ที่ {{ช่องทางติดต่อ}} ทีมงานพร้อมดูแลค่ะ
```
