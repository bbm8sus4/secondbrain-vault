## จังหวะ 1 — รับทราบปัญหา (PRE-APPROVED ส่งได้ทันที)

> ห้ามแก้โครงข้อความ เติมได้เฉพาะช่อง `{{...}}` — ส่งแล้วแจ้ง C ทาง LINE ทันที

### LINE OA

```
[{{แบรนด์}}] แจ้งเหตุขัดข้องค่ะ

ขณะนี้พบว่า {{อาการ}} ตั้งแต่เวลา {{เวลาเริ่มพบ}}
ทีมงานรับทราบปัญหาแล้ว และกำลังเร่งตรวจสอบแก้ไขอย่างเร่งด่วนค่ะ

จะแจ้งความคืบหน้าให้ทราบเป็นระยะ ขออภัยในความไม่สะดวกค่ะ
```

### SMS (ใช้เมื่อกระทบหนัก/เกิน 1 ชม.) — ≤ 70 ตัวอักษร

```
{{แบรนด์}}: ระบบขัดข้องชั่วคราว ทีมงานกำลังเร่งแก้ไข ขออภัยค่ะ
```

## จังหวะ 2 — อัปเดตระหว่างแก้ (ผ่าน C ก่อนส่ง)

> ส่งทุก 30–60 นาทีถ้าเหตุยังไม่จบ — การเงียบทำให้ลูกค้าคิดว่าเราไม่ทำอะไร

### LINE OA

```
[{{แบรนด์}}] อัปเดตความคืบหน้าค่ะ

{{สถานะปัจจุบัน เช่น ทีมงานพบจุดที่เป็นปัญหาแล้ว กำลังดำเนินการแก้ไข}}
คาดว่าจะกลับมาใช้งานได้ภายในเวลา {{เวลาคาดการณ์}}

ระหว่างนี้ {{คำแนะนำชั่วคราวถ้ามี เช่น สามารถตรวจสลิปย้อนหลังได้เมื่อระบบกลับมา ยอดไม่หาย}}
ขออภัยในความไม่สะดวกค่ะ
```

### LINE OA

```

สรุปเหตุการณ์: {{อาการ}} ระหว่างเวลา {{เวลาเริ่มพบ}}–{{เวลากลับมาปกติ}}

ขณะนี้ทีมงานแก้ไขเรียบร้อย และ{{มาตรการกันเกิดซ้ำแบบย่อ}}แล้วค่ะ

{{ถ้ามีชดเชย: เพื่อเป็นการขออภัย ทีมงานขอมอบ {{รายละเอียดการชดเชย}} ให้ลูกค้าทุกท่านที่ได้รับผลกระทบ โดย {{วิธีรับ/มีผลอัตโนมัติ}} ค่ะ}}

ขออภัยอย่างยิ่งในความไม่สะดวกที่เกิดขึ้น

```

## หมายเหตุ

- เหตุที่ **กระทบเงิน/ยอดลูกค้า** (เช่น สลิปซ้ำ ยอดหาย) — จังหวะ 3 ต้องบอกชัดว่ายอด/แต้มของลูกค้าได้รับการแก้กลับครบ
- การชดเชย (เครดิต/วันใช้งานฟรี) — C ตัดสินใจเท่านั้น CS เสนอได้แต่ห้ามสัญญากับลูกค้าก่อน
