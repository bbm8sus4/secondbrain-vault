## วิธีใช้ 3 ขั้น

1. **เลือกเทมเพลต** ตามประเภทงาน → เปิดไฟล์ 10–13 คัดลอกเวอร์ชันที่ต้องใช้ แล้วเติมช่อง `{{...}}`
2. **เช็คก่อนส่ง** → ไล่ [[20-Checklist-ก่อนส่ง]] ให้ครบทุกข้อ + ขออนุมัติตามขั้นใน [[00-SOP-ประกาศลูกค้า]]
3. **ส่งตามช่องทาง** ที่กำหนดใน [[01-Channel-Matrix]] → บันทึกลง [[21-Log-ประกาศ]]

## กติกาสั้นที่สุด (จำ 3 ข้อ)

1. **ทุกประกาศต้องผ่าน C อนุมัติ** — ยกเว้นข้อความ "รับทราบปัญหา" ตอนเหตุขัดข้อง (pre-approved แล้ว ดู [[11-เทมเพลต-เหตุขัดข้อง]])
2. **ด่วน = SMS ก่อน, ไม่ด่วน = LINE** — เหตุผลอยู่ใน [[01-Channel-Matrix]]
3. **ส่งแล้วต้องลง log ทันที** — ไม่ลง log = เหมือนไม่ได้ส่ง
