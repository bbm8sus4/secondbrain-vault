# 10+ Prompts ChatGPT Image 2.0 (KEM LIFE) — Takeaways

> **TL;DR:** คลัง prompt สำเร็จรูป 10 แบบสำหรับ ChatGPT Image 2.0 — เด่นสุดคือ "วาง URL สินค้า/บทความ → ได้ภาพโฆษณา/infographic จบใน prompt เดียว" เพราะโมเดลไปอ่านข้อมูลจากลิงก์เองก่อนเจนภาพ

Raw: [[03 Resources/Clippings/เปิดคลังแจก 10+ วิธีใช้ ChatGPT Image 2.0 ฟรี!!!|ดูตัวจริง (KEM LIFE)]]

## Takeaways สำหรับ COO

1. **URL → ภาพโฆษณา ใน prompt เดียว** — วางลิงก์หน้าสินค้า/เว็บ แล้วโมเดลดึงข้อมูลจริง (ชื่อ, ราคา, โลโก้) มาจัดเป็น ad ให้เอง → ใช้กับหน้า product ของ EasySlip/BoostSMS ทำภาพประกอบโพสต์/broadcast ได้ทันที ไม่ต้องบรีฟใหม่
2. **URL บทความ → infographic** — วางโครงไว้ให้ (หัวข้อ / ปัญหา / ประเด็นสำคัญ / สรุปท้ายภาพ) แล้วให้โมเดลเติมเนื้อจากลิงก์ ภาษาไทยตรง ไม่เบี้ยว → เหมาะแปลงประกาศ/คู่มือ feature เป็นภาพเดียวส่งใน LINE OA
3. **เจนปก/thumbnail ทีเดียว 6 แบบแล้วค่อยคัด** — ให้ทีมเลือกจากตัวเลือกจริงแทนการจินตนาการ ลดรอบแก้งาน — pattern นี้ใช้ได้กับ artwork ทุกประเภท (เจนหลายแบบ → คัด → refine อันเดียว)
4. **คุมสีแบรนด์ด้วย hex code** — ระบุโค้ดสีของแบรนด์ลงใน prompt ตรง ๆ → ทำ template prompt ประจำแบรนด์ (Thunder/EasySlip/KuanGolf) ที่ล็อกสี+font ไว้ ให้ใครในทีมเจนก็ได้ธีมเดียวกัน
5. **ภาพชุด e-commerce จากลิงก์สินค้า + แคปหน้าจอกันเพี้ยน** — แนบ screenshot สินค้าจริงไปพร้อม prompt เพื่อกันโมเดลมโนหน้าตาสินค้า ได้ภาพชุด 6 รูปแบบมืออาชีพ
6. **Mockup App Store screenshot สำหรับขายงาน** — เจนภาพหน้าจอแอปจากแค่ชื่อ+จุดขาย+กลุ่มลูกค้า → ใช้ทำ mockup เสนอลูกค้า/pitch feature ใหม่ก่อนลงมือพัฒนาจริง (เช่น หน้าตา dashboard EasyCRM เวอร์ชันถัดไป)
7. **ภาพโครงสร้างหลักสูตร/คอร์ส** — สรุปคอร์สเป็นภาพเดียวสำหรับ landing page → ใช้กับเอกสารอบรม AI ภายในองค์กร หรือ onboarding ลูกค้า
8. **การ์ตูน How-to หลายช่องพร้อมบทพูด** — เล่าขั้นตอนการใช้งานเป็นการ์ตูนต่อเนื่อง ตัวหนังสืออ่านได้จริง → ทางเลือกทำสื่อสอนลูกค้าใช้ EasySlip/EasyBOT แบบย่อยง่าย
9. **หลักการสั่งงาน: กำหนดแค่โครงใหญ่ อย่าลงดีเทลจนไปกดความฉลาดของโมเดล** — บอกโครงสร้าง สไตล์ ข้อห้าม แล้วปล่อยให้มันเติมรายละเอียดเอง มักได้ผลดีกว่าบรีฟละเอียดยิบ
