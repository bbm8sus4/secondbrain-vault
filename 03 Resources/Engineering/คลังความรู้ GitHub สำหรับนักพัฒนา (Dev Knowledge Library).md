# คลังความรู้ GitHub สำหรับนักพัฒนา

รวมของดีที่คนทั้งโลกช่วยกันแชร์บน GitHub — แนวเดียวกับ [[Clean Code (Robert C. Martin) — สรุป|Clean Code]] ที่เพิ่งเก็บไป กดลิงก์เข้าไปอ่าน/โหลดได้เลย จัดเป็นหมวดตามการใช้งาน

---

## 1. Gist อ่านจบในหน้าเดียว (แบบเดียวกับ Clean Code)

ของสั้น กระชับ เปิดปุ๊บได้ของ ไม่ต้องอ่านทั้งเล่ม

- **[Clean Code Summary — wojteklu](https://gist.github.com/wojteklu/73c6914cc446146b8b533c0988cf8d29)** — สรุปหนังสือ Clean Code ทั้งเล่มเหลือหน้าเดียว *(เก็บ verbatim ไว้แล้วที่ [[Clean Code (Robert C. Martin) — สรุป]])*
- **[System Design Cheatsheet — vasanthk](https://gist.github.com/vasanthk/485d1c25737e8e72759f)** — สูตรออกแบบระบบใหญ่ ตั้งแต่รับ traffic เยอะ ๆ, cache, load balancer, database ยันขั้นตอนคิดตอนสัมภาษณ์
- **[Git CLI Cheatsheet — hofmannsven](https://gist.github.com/hofmannsven/6814451)** — คำสั่ง git ที่ใช้จริงบ่อย ๆ รวมไว้ที่เดียว
- **[awesome-gists — sahilbansal17](https://github.com/sahilbansal17/awesome-gists)** ★~2.9k — **ตัวแม่**: รายชื่อ gist ดัง ๆ ที่คนคัดมาให้ (ถ้าชอบแนว gist กดอันนี้ก่อนเลย)

## 2. เขียนโค้ดให้สะอาด / Best Practices

- **[clean-code-javascript — ryanmcdermott](https://github.com/ryanmcdermott/clean-code-javascript)** ★~94k — เอาหลัก Clean Code มาปรับให้เข้ากับ JavaScript พร้อมตัวอย่างก่อน/หลัง
- **[nodebestpractices — goldbergyoni](https://github.com/goldbergyoni/nodebestpractices)** ★~105k — 80+ best practices ของ Node.js ตั้งแต่โครงสร้างโปรเจกต์ ยัน error handling/security
- **[Front-End-Checklist — thedaviddias](https://github.com/thedaviddias/Front-End-Checklist)** ★~69k — เช็กลิสต์ก่อนปล่อยเว็บขึ้น production (HTML/SEO/performance/security)
- **[airbnb/javascript](https://github.com/airbnb/javascript)** ★~146k — JavaScript Style Guide ที่บริษัททั่วโลกใช้เป็นมาตรฐาน
- **[google/styleguide](https://github.com/google/styleguide)** ★~38k — style guide ของ Google ครบทุกภาษา (C++, Python, Java, Go, ฯลฯ)

## 3. System Design / ออกแบบระบบใหญ่

- **[system-design-primer — donnemartin](https://github.com/donnemartin/system-design-primer)** ★~352k — คัมภีร์ออกแบบระบบสเกลใหญ่ + เตรียมสัมภาษณ์ มีทั้งทฤษฎีและ Anki flashcards
- **[awesome-scalability — binhnguyennus](https://github.com/binhnguyennus/awesome-scalability)** ★~63k — รวมบทความ/เคสจริงเรื่อง scalability, availability, performance จากบริษัทเทคใหญ่

---

## 4. เรียน / เตรียมสัมภาษณ์ / Roadmap

- **[developer-roadmap — kamranahmedse](https://github.com/kamranahmedse/developer-roadmap)** ★~320k — แผนที่เส้นทางเป็นสายอาชีพ (frontend/backend/devops/AI) ดูรูปเดียวรู้ว่าต้องเรียนอะไรต่อ
- **[coding-interview-university — jwasham](https://github.com/jwasham/coding-interview-university)** ★~320k — แผนอ่านหนังสือ/เรียน CS ตั้งแต่ศูนย์จนพร้อมสัมภาษณ์บริษัทใหญ่
- **[tech-interview-handbook — yangshun](https://github.com/yangshun/tech-interview-handbook)** ★~130k — คู่มือสัมภาษณ์งานเทค เนื้อหาอยู่ในเล่มเลย ไม่ต้องกระโดดไปที่อื่น
- **[javascript-algorithms — trekhleb](https://github.com/trekhleb/javascript-algorithms)** ★~193k — algorithm + data structure เขียนด้วย JS มีอธิบายทุกตัว
- **[project-based-learning — practical-tutorials](https://github.com/practical-tutorials/project-based-learning)** ★~230k — เรียนด้วยการสร้างของจริง (build your own X) แยกตามภาษา

## 5. คลังใหญ่ / Awesome Lists (เมกะรวมทุกอย่าง)

- **[build-your-own-x — codecrafters-io](https://github.com/codecrafters-io/build-your-own-x)** ★~521k — สอนสร้างเองตั้งแต่ศูนย์: database, git, OS, browser, docker ฯลฯ (repo ดาวเยอะสุดในโลก)
- **[awesome — sindresorhus](https://github.com/sindresorhus/awesome)** ★~480k — ต้นตำรับ "awesome list" ประตูสู่ทุกหัวข้อ อยากเรียนอะไรมาที่นี่ก่อน
- **[public-apis](https://github.com/public-apis/public-apis)** ★~446k — รวม API ฟรีทุกหมวด เอาไปต่อยอดโปรเจกต์
- **[free-programming-books — EbookFoundation](https://github.com/EbookFoundation/free-programming-books)** ★~391k — หนังสือ/คอร์สโปรแกรมมิ่งฟรี แยกตามภาษา (มีภาษาไทยด้วย)
- **[30-seconds-of-code — Chalarangelo](https://github.com/Chalarangelo/30-seconds-of-code)** ★~123k — snippet สั้น ๆ เข้าใจใน 30 วินาที คัดลอกไปใช้ได้เลย
- **[awesome-cheatsheets — LeCoupa](https://github.com/LeCoupa/awesome-cheatsheets)** ★~41k — cheatsheet ภาษา/framework/tool รวมไว้ไฟล์เดียวต่ออย่าง

## อยากได้แบบ Clean Code (เก็บ verbatim ลงเครื่อง)?

ตอนนี้ที่โหลดเนื้อหาเต็มมาเก็บในวอลต์แล้วมีแค่ Clean Code ตัวอื่นในลิสต์นี้เก็บเป็น "ลิงก์" ไว้ กดเข้าไปอ่านบนเว็บได้ — ถ้าอยากให้ดึงตัวไหนมาเก็บแบบ verbatim (เช่น System Design Cheatsheet, Git Cheatsheet) บอกชื่อมาได้เลย เดี๋ยวจัดให้เหมือนกัน
