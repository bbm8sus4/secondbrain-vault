## มิติที่ 4 — วิธี Offboarding ที่ถูกต้อง (กฎหมายแรงงานไทย)

> พนักงานทดลองงาน = ลูกจ้างเต็มตัวตั้งแต่วันแรก · ยืนยันตัวเลข/สิทธิกับฝ่ายบัญชี/ที่ปรึกษาแรงงานอีกที (ขึ้นกับสัญญา+อายุงานจริง)

**ก. เอกสาร:** รับหนังสือลาออก + ระบุวันสุดท้าย · ยืนยันลาออกเอง(ไม่ต้องจ่ายชดเชย) vs เลิกจ้างร่วมกัน · notice ตามสัญญา
**ข. เงิน:** จ่ายค่าจ้างถึงวันสุดท้าย (เลิกจ้างต้องจ่ายใน 3 วัน ม.70) · จ่ายวันลาพักร้อนคงเหลือตามสัดส่วน (ทดลองงานอาจยังไม่ครบสิทธิ เช็คนโยบาย)
**ค. ราชการ:** แจ้งออกประกันสังคม (สปส.6-09) ภายในวันที่ 15 เดือนถัดไป · ออก 50 ทวิ · หนังสือรับรองการทำงาน (ถ้าขอ)
**ง. ทรัพย์สิน/ข้อมูล:** คืนของ + เพิกถอน access + PDPA (ห้ามเก็บ copy ข้อมูลพนักงาน)
**จ. ปิด:** exit interview (สำคัญ — เธอทดลองงาน+มีเรื่องผลงาน)

# 4. หนังสือรับรอง + ประกาศแจ้งทีม

### 4.1 หนังสือรับรองการทำงาน
```
                                          บริษัท [ธันเดอร์ โซลูชั่น จำกัด]
                                          วันที่ [__ ____ พ.ศ. ____]

                          หนังสือรับรองการทำงาน  เลขที่ [HR-CERT-____]

     หนังสือฉบับนี้ให้ไว้เพื่อรับรองว่า [ชื่อ-นามสกุล]
เลขประจำตัวประชาชน [______] ได้เข้าทำงานกับบริษัทฯ
ในตำแหน่ง [เจ้าหน้าที่ฝ่ายทรัพยากรบุคคล]
ตั้งแต่วันที่ [__ ____] ถึงวันที่ [__ ____]
     บริษัทฯ ออกหนังสือฉบับนี้ให้ไว้เพื่อเป็นหลักฐาน

                                          ลงชื่อ ______________________
                                          ( [ชื่อผู้มีอำนาจ] )  ตำแหน่ง [COO]
                                          [ตราประทับบริษัท]
```
> คู่กัน: **50 ทวิ** (ฝ่ายบัญชีออก)

### 4.2 Email/ประกาศแจ้งทีม
**หัวข้อ:** แจ้งการเปลี่ยนแปลงผู้รับผิดชอบงาน HR
```
เรียนทีมงานทุกท่าน
ขอแจ้งว่า คุณ[ต้า] จะปฏิบัติงานวันสุดท้ายในวันที่ [__ ____]
ขอขอบคุณสำหรับการทำงานที่ผ่านมา

ตั้งแต่วันที่ [__ ____] งานด้าน HR ทั้งหมด (ลา สรรหา สวัสดิการ เอกสาร) ติดต่อ:
    คุณป๊อบ (ยุพาพร คำสีแก้ว) — HR Manager · ช่องทาง: [___]

หากมีงานค้างกับคุณ[ต้า] กรุณาแจ้งคุณป๊อบภายใน [__ ____]
ขอบคุณครับ · [Bob/ฝ่ายบริหาร]
```

### 4.3 ข้อความแจ้งผู้สมัครที่ค้าง
```
เรียน คุณ[ชื่อผู้สมัคร]
ผู้ประสานงานสัมภาษณ์มีการเปลี่ยนแปลง ต่อจากนี้คุณป๊อบจะดูแลและติดต่อกลับ
ขออภัยในความไม่สะดวก และขอบคุณที่สนใจร่วมงานครับ
```

### 4.4 หนังสือตอบรับการลาออก
```
ตามที่ท่านยื่นหนังสือลาออกลงวันที่ [__ ____] กำหนดวันสุดท้าย [__ ____]
บริษัทฯ รับทราบและอนุมัติแล้ว ขอให้ส่งมอบงานตามแบบฟอร์มและคืนทรัพย์สินให้ครบก่อนวันสุดท้าย
บริษัทฯ จะจ่ายค่าจ้างงวดสุดท้ายและสิทธิประโยชน์คงเหลือตามระเบียบและกฎหมาย
                                          ลงชื่อ ____________ ฝ่ายบริหาร/HR
```
