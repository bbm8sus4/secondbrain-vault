# HR Templates — หนังสือรับรองการทำงาน + ประกาศแจ้งทีม

> เทมเพลตกลาง ใช้ซ้ำได้ทุกครั้งที่มีคนออก · เติมช่อง [ ] · ออกโดย POP (HR) เสนอ Bob เซ็น

---

## 1. หนังสือรับรองการทำงาน (Certificate of Employment)

> ออกให้เมื่อพนักงานร้องขอ · เนื้อหาเป็นกลาง ระบุแค่ตำแหน่ง+ช่วงเวลา (ไม่ต้องระบุเหตุออกหรือผลงาน)

```
                                          บริษัท [ธันเดอร์ โซลูชั่น จำกัด]
                                          [ที่อยู่บริษัท]
                                          วันที่ [__ เดือน ____ พ.ศ. ____]

                          หนังสือรับรองการทำงาน
                          เลขที่ [HR-CERT-____]

     หนังสือฉบับนี้ให้ไว้เพื่อรับรองว่า [ชื่อ-นามสกุล]
เลขประจำตัวประชาชน [______________] ได้เข้าทำงานกับบริษัทฯ
ในตำแหน่ง [เจ้าหน้าที่ฝ่ายทรัพยากรบุคคล / HR Officer]
ตั้งแต่วันที่ [__ ____ ____] ถึงวันที่ [__ ____ ____]

     ในระหว่างการปฏิบัติงาน ได้ปฏิบัติหน้าที่ [ด้วยความรับผิดชอบ / ตามที่ได้รับมอบหมาย]

     บริษัทฯ ออกหนังสือฉบับนี้ให้ไว้เพื่อเป็นหลักฐาน

                                          ลงชื่อ ______________________
                                          ( [ชื่อผู้มีอำนาจ] )
                                          ตำแหน่ง [COO / กรรมการ]
                                          [ตราประทับบริษัท]
```

> เอกสารคู่กันที่ต้องออกให้ด้วย: **50 ทวิ** (หนังสือรับรองหักภาษี ณ ที่จ่าย) — ฝ่ายบัญชีออก

---

## 2. Email / ประกาศแจ้งทีม (ภายใน)

> แจ้งสั้น กระชับ สุภาพ — บอกว่าใครดูแล HR ต่อ + ส่งงานค้างไปที่ไหน · ไม่ต้องระบุเหตุผลการลาออก

**หัวข้อ:** แจ้งการเปลี่ยนแปลงผู้รับผิดชอบงาน HR

```
เรียนทีมงานทุกท่าน

ขอแจ้งว่า คุณ[ต้า/Guitar] จะปฏิบัติงานวันสุดท้ายในวันที่ [__ ____ ____]
ขอขอบคุณสำหรับการทำงานที่ผ่านมา และขออวยพรให้ประสบความสำเร็จในเส้นทางต่อไป

ตั้งแต่วันที่ [__ ____ ____] เป็นต้นไป งานด้านทรัพยากรบุคคลทั้งหมด
(การลา สรรหา สวัสดิการ เอกสารพนักงาน) กรุณาติดต่อ:
    คุณป๊อบ (ยุพาพร คำสีแก้ว) — HR Manager
    ช่องทาง: [LINE/เบอร์/กลุ่ม]

หากท่านมีงานหรือเรื่องที่ค้างอยู่กับคุณ[ต้า] กรุณาแจ้งคุณป๊อบภายในวันที่ [__ ____]
เพื่อส่งต่อให้เรียบร้อย

ขอบคุณครับ
[Bob / ฝ่ายบริหาร]
```

**ประกาศย่อสำหรับผู้สมัครที่ค้างในสาย (ถ้าจำเป็น):**
```
เรียน คุณ[ชื่อผู้สมัคร]
ขอเรียนว่าผู้ประสานงานสัมภาษณ์ของท่านมีการเปลี่ยนแปลง
ต่อจากนี้ คุณป๊อบจะเป็นผู้ดูแลและติดต่อกลับ ขออภัยในความไม่สะดวก
และขอบคุณที่ให้ความสนใจร่วมงานกับเราครับ
```

## 3. หนังสือตอบรับการลาออก (Acknowledgement) — ออกให้พนักงาน

```
ตามที่ท่านได้ยื่นหนังสือลาออกลงวันที่ [__ ____ ____]
โดยกำหนดวันทำงานวันสุดท้ายในวันที่ [__ ____ ____]

     บริษัทฯ ได้รับทราบและอนุมัติการลาออกของท่านแล้ว
โดยขอให้ท่านส่งมอบงานตามแบบฟอร์มส่งมอบงาน และคืนทรัพย์สินของบริษัทฯ
ให้ครบถ้วนก่อนวันทำงานวันสุดท้าย

     บริษัทฯ จะดำเนินการจ่ายค่าจ้างงวดสุดท้ายและสิทธิประโยชน์คงเหลือ
ตามระเบียบและกฎหมายที่เกี่ยวข้อง

                                          ลงชื่อ ______________________
                                          ฝ่ายบริหาร / HR
```

*ทุกฉบับมี PII — เก็บเป็นความลับ ห้าม deploy public · ยืนยันถ้อยคำทางกฎหมายกับที่ปรึกษาแรงงานได้ตามสะดวก*
