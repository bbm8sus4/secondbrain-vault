# คลัง Prompt Repos ดัง (GitHub)

repo พรอมป์/prompt engineering ที่คนทั้งโลกดาวเยอะสุด — ต่างจาก[[README|Prompt Library]] ของเรา (พรอมป์ที่พี่สร้างเอง) อันนี้คือ**ของคนอื่นบน GitHub** เอาไว้กดเข้าไปหยิบพรอมป์/เรียนรู้เพิ่ม

---

## 1. คลังพรอมป์พร้อมใช้ (copy ไปแปะได้เลย)

- **[f/awesome-chatgpt-prompts (prompts.chat)](https://github.com/f/awesome-chatgpt-prompts)** ★~143k — คลังพรอมป์ชุมชนที่ใหญ่/ดังสุดในโลก ใช้ได้ทุกโมเดล (ChatGPT/Claude/Gemini) มีเว็บ prompts.chat ให้ค้น
- **[ai-boost/awesome-prompts](https://github.com/ai-boost/awesome-prompts)** — คัดพรอมป์จาก GPTs อันดับต้น ๆ ใน GPT Store + prompt attack/protect
- **[abilzerian/LLM-Prompt-Library](https://github.com/abilzerian/LLM-Prompt-Library)** — พรอมป์ทดลอง + Jinja2 templates หลายโมเดล (OpenAI/Anthropic/DeepSeek/Mistral/xAI)
- **[danielrosehill/Prompt-Library](https://github.com/danielrosehill/Prompt-Library)** — คลังพรอมป์ส่วนตัวพร้อมตัวอย่าง output + โน้ต

## 2. เรียนรู้ Prompt Engineering (คู่มือ/ทฤษฎี)

- **[dair-ai/Prompt-Engineering-Guide](https://github.com/dair-ai/Prompt-Engineering-Guide)** ★~55k — คัมภีร์เรียน prompt engineering ตั้งแต่ zero-shot ยัน tree-of-thought / self-consistency / RAG / AI agents · มี 13 ภาษา
- **[promptslab/Awesome-Prompt-Engineering](https://github.com/promptslab/Awesome-Prompt-Engineering)** — รวม resource/เปเปอร์/เครื่องมือ prompt engineering แบบคัดมือ
- **[brexhq/prompt-engineering](https://github.com/brexhq/prompt-engineering)** — คู่มือ + งานวิจัยเชิงลึกจากทีม Brex

## 3. System Prompts / ของหลุดจากเครื่องมือดัง (ทองคำสำหรับสาย AI)

ดูว่าเครื่องมือ AI ระดับโลกเขาเขียน system prompt กันยังไงจริง ๆ — เรียนลัดจากมืออาชีพ

- **[x1xhlol/system-prompts-and-models-of-ai-tools](https://github.com/x1xhlol/system-prompts-and-models-of-ai-tools)** ★~137k — system prompt จริงของ 28+ เครื่องมือ: Cursor, Claude Code, Windsurf, v0, Devin, Lovable, Perplexity, Warp ฯลฯ (มี tool schema JSON ด้วย)
- **[jujumilk3/leaked-system-prompts](https://github.com/jujumilk3/leaked-system-prompts)** — คลัง leaked system prompts รวมหลายเจ้า
- **[0xeb/TheBigPromptLibrary](https://github.com/0xeb/TheBigPromptLibrary)** — รวม system prompts + custom instructions + jailbreak + protection prompts (ChatGPT/Copilot/Claude/Gemini)
- **[asgeirtj/system_prompts_leaks](https://github.com/asgeirtj/system_prompts_leaks)** — system prompt ของ Claude/ChatGPT/Gemini/Grok อัปเดตเรื่อย ๆ
