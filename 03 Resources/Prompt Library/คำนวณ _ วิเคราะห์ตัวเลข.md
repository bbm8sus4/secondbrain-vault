# คำนวณ / วิเคราะห์ตัวเลข

**คำนวณ** **/ วิเคราะห์ตัวเลข**

**\**

ทำได้ แต่ต้องสั่งให้ “แสดงสมมติฐาน + ขั้นตอน + ตรวจความสมเหตุสมผล”

- “คำนวณกำไรขั้นต้น, กำไรสุทธิ จากข้อมูลนี้… และสรุปเป็นตาราง”
- “คำนวณ MoM growth รายเดือน + เฉลี่ย 3 เดือนย้อนหลัง”
- “ทำ break-even: ต้นทุนคงที่/ผันแปร, จำนวนที่ต้องขายต่อวัน”
- “ช่วยตรวจตัวเลขว่ามีจุดผิดปกติไหม (outliers / หน่วยผิด / รวมผิด)”

**คำสั่งกันพลาด (แนะนำใช้ทุกครั้ง):**“ก่อนสรุป ให้คุณ 1) ระบุหน่วย 2) ระบุสมมติฐาน 3) แสดงสูตร 4) ทำ sanity check 3 ข้อ”
