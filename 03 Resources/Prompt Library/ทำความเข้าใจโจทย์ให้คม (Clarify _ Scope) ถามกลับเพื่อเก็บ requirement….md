# ทำความเข้าใจโจทย์ให้คม (Clarify / Scope)  ถามกลับเพื่อเก็บ requirement…

**ทำความเข้าใจโจทย์ให้คม** **(Clarify / Scope)**ถามกลับเพื่อเก็บ requirement: “ถามฉัน 10 คำถามเพื่อทำให้โจทย์ชัด”

แยกสิ่งที่ “รู้/ไม่รู้/ต้องตัดสินใจ”: “ทำรายการสิ่งที่ยังขาด และวิธีหาให้ได้”

แปลงโจทย์กำกวมเป็นสเปก: “สรุปเป็น requirement + acceptance criteria”
