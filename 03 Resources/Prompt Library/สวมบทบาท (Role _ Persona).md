# สวมบทบาท (Role / Persona)

**สวมบทบาท** **(Role / Persona)**

**\**

ใช้เพื่อ “ได้มุมมอง” ไม่ใช่เพื่อเชื่อแบบตาบอด

- “สวมบทบาทเป็น **COO** ช่วยทำแผน 90 วันสำหรับโปรเจกต์นี้ พร้อม KPI และความเสี่ยง”
- “สวมบทบาทเป็น **ที่ปรึกษากฎหมายธุรกิจ** ช่วยเช็กลิสต์ประเด็นที่ต้องระวัง (ไม่ต้องให้คำแนะนำทางกฎหมายเชิงยืนยัน)”
- “สวมบทบาทเป็น **ผู้ตรวจสอบคุณภาพ (QA)** หา edge case และข้อผิดพลาดที่อาจเกิดขึ้น 20 ข้อ”
- “สวมบทบาทเป็น **CFO** ตรวจงบประมาณ: รายการไหนบานปลายง่าย + วิธีคุม”

**ทริก:** สั่ง “ให้โต้แย้งตัวเอง” เช่น “ให้คุณเสนอคำตอบ แล้วแย้งคำตอบนั้นอีก 5 ข้อ พร้อมทางแก้”
