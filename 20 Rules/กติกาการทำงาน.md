# กติกาการทำงานของ AI

> AI Agent ทุกตัวที่อ่าน vault นี้ ต้องทำตามกติกานี้

## ภาษา
- ตอบภาษาไทยธรรมชาติ สั้น กระชับ ลงท้าย "ครับ"
- คงภาษาอังกฤษ: โค้ด, คำสั่ง, path, error message, ศัพท์เทคนิค

## การสร้างโน้ต
- **ห้ามสร้างโน้ตซ้ำ** — ค้นก่อนว่ามีอยู่แล้วไหม ถ้ามีให้อัพเดตไฟล์เดิม
- โน้ตใหม่ที่ยังไม่รู้จะเก็บไหน → `00 Inbox/` เท่านั้น
- ทุกโน้ตเชื่อมโยงด้วย `[[wikilink]]` ถ้าเกี่ยวข้องกัน

## โฟลเดอร์ต้องห้าม
- `30 Claude Memory/` — **read-only** เป็น mirror อัตโนมัติ ห้ามแก้ในนี้
  (แก้ที่ต้นทาง `~/.claude-warp/projects/-Users-aexgee/memory/` แทน — ชื่อไฟล์ต้นทางเป็นอังกฤษ ดู mapping ใน `~/bin/sync-memory-to-vault.py`)

## Deploy / งานภายนอก
- Cloudflare Pages: `git push` อย่างเดียว แชร์ stable URL ไม่แชร์ per-commit hash
- Mass-send (LINE OA / broadcast): pacing ช้า 3-5s+, snapshot ลงดิสก์, เช็ค already-sent ก่อน resume

## ส่งมอบงาน
- สรุปผลก่อน รายละเอียดทีหลัง อ่านจบบนมือถือได้
- งานเสร็จ → log ลง daily note หรือ project note ทุกครั้ง ไม่ให้ Agent ตัวถัดไปเดาใหม่
