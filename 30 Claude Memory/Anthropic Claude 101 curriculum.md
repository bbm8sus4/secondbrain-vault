Anthropic Academy's Claude 101 (free, ~2-3 hrs) — 5 modules:

1. **Meet Claude** — What is Claude · First conversation · Getting better results · Desktop app (Chat/Cowork/Code)
2. **Organizing work** — Projects · Artifacts · Skills
3. **Expanding reach** — Tools · Enterprise search · Research mode
4. **Putting it together** — Use-cases by role · Other ways
5. **Certificate**

**Key insights for workshop design:**
- Anthropic does NOT teach a prompt framework (no CRAFT/CRIT). Uses "Getting better results" + examples.
- Killer features they emphasize but easy to skip: **Artifacts** (canvas), **Projects** (persistent context), **Skills** (reusable prompts)
- Top 5 prompt techniques: XML tags · ask for evidence · assign roles · step-by-step reasoning · show examples
- Full curriculum + gap analysis vs Khonkaen workshop: `~/SecondBrain/03 Resources/AI Workshops/Reference-Anthropic-Claude-101-Curriculum.md`
- GitHub repo `anthropics/courses` = 5 dev-track courses (API fundamentals, Prompt eng interactive, Real world prompting, Prompt evaluations, Tool use)

Learning paths after 101: → AI Fluency (beginner), Building with Claude API (dev), AI Fluency for Educators.
