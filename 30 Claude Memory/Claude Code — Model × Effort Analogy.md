Model × Effort analogy สำหรับสอน Claude Code อยู่ที่ `~/SecondBrain/03 Resources/AI Workshops/Claude Code — Model × Effort อธิบายแบบเปรียบเทียบ.md`

Core analogy:
- **Model = เครื่องยนต์**: Fable=F1, Opus=V8, Sonnet=Turbo 4 สูบ, Haiku=มอไซค์
- **Effort = คันเร่ง**: low→max = เหยียบนิดเดียว→red line
- **สูตร**: ผลลัพธ์ = เครื่องยนต์ × คันเร่ง
- **Key insight**: Sonnet 5 + high ≈ Opus 4.8 + medium

รวม: สูตรเลือกตามงาน, ตามงบ, เทคนิคผสม model ต่างกันใน multi-agent, ASCII chart
