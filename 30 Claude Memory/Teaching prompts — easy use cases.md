Ready-to-use **teaching prompt collection for beginners** — copy-paste prompts grouped by use case, built for COO's AI workshops. Teaches good-prompt anatomy (บริบท-งาน-รูปแบบ-ข้อจำกัด) then gives ~20 prompts across 5 categories:
1. HTML apps (income/expense tracker = flagship, to-do, habit tracker, invoice/quotation, BMI calc, savings goal)
2. Office/docs (meeting summary, formal email, translate, doc summary, SOP)
3. Marketing/content (30-day ideas, captions, ad copy A/B, FAQ)
4. Data/analysis (analyze Excel/CSV, HTML dashboard, charts)
5. Learning (ELI5, quiz, learning plan)
