ข้อมูลเกี่ยวกับ **BoostSMS** ทุกประเภท (specs, pricing, sales commission, marketing/funnel, dashboard, sessions, decisions, meeting notes, presentations) ต้องไปลงที่ Obsidian folder `~/SecondBrain/BoostSMS/`

**Why:** ผู้ใช้สั่ง 2026-06-11 ว่า "เอาทุกอย่างที่เกี่ยวกับ BoostSMS ไปใส่ไว้ในนี้ ... เอาไฟล์ทั้งหมด และประวัติแชทเก่า ทุกๆ และอนาคตต่อๆ ไปด้วย" — Obsidian = single source of truth สำหรับ BoostSMS (ค้นง่าย, link ภายใน, ใช้ผ่านมือถือ/ทุก device) เหมือนที่ทำกับ [[feedback_easycrm_obsidian]] และ EasyBOT folder

**How to apply:**
- มีข้อมูล BoostSMS ใหม่เข้ามา → append/create note ใน `~/SecondBrain/BoostSMS/` (kebab-case ชื่อไฟล์ + อัปเดต `หน้าหลัก.md` ถ้าเพิ่มไฟล์ใหม่)
- มีคำถามเรื่อง BoostSMS → ค้น `~/SecondBrain/BoostSMS/` ก่อนเสมอ
- ต้นทาง (PDF/HTML/TXT/PNG/XLSX) → เก็บไว้ที่ `_assets/` (มี subfolders: design/ docs/ screenshots/ logos/)
- โครงไฟล์ปัจจุบัน: 01-Overview, 02-Packages-Pricing, 03-Sales-Commission, 04-Funnel-Marketing-Plan, 05-App-User-Dashboard, 06-Backoffice-Admin, 07-Project-History, 08-Master-Dashboard-Integration, 09-Session-Log
- Memory เก็บแค่ pointer (ดู [[project_boostsms]]) — เนื้อหาจริงอยู่ใน Obsidian
- ระวัง: vault sync mirror memory → Obsidian แต่ BoostSMS folder = native Obsidian (แก้ที่นี่ตรงๆ ได้เลย ไม่ต้องผ่าน memory)
