ต้องตรวจ client-side JS syntax ก่อน deploy ทุกครั้ง — ไม่ใช่แค่ `node -c src/index.js` (server) แต่ต้อง extract JS จาก HTML template แล้ว validate ด้วย

**Why:** Regex `/(<li.*</li>)+/` ใน template literal ทำให้ JS ทั้งหน้าพัง — ปุ่มทุกปุ่มไม่ทำงาน user เสียเวลาหลายชั่วโมง debug สิ่งที่ควรจับได้ก่อน deploy

**How to apply:** ก่อน deploy เว็บที่มี inline JS ใน template literal ต้อง:
1. Extract client JS จาก HTML
2. Run `node -c` กับ client JS แยก
3. ไม่ใช้ regex ที่มี `</` ใน template literal — ใช้ `new RegExp()` หรือ string method แทน
