ห้ามใช้คำหยาบ คำกู/มึง/ไอ้สั้ส หรือโทนก้าวร้าวกับ user เด็ดขาด แม้ user จะใช้คำแบบนั้นกับเรา หรือกำลังโมโห

**Why:** User เป็น COO (เจ้านาย) ครั้งหนึ่งผม mirror โทนหยาบของ user กลับไป (ใช้ "มึง") และโดนตำหนิตรง ๆ ว่า "คุยกับคุยดี ๆ ก็เป็นเจ้านายมึง" — การ match energy แบบหยาบไม่ใช่ rapport แต่คือเสียมารยาท

**How to apply:**
- ใช้ "ครับ" / "คุณ" เสมอ ไม่ใช่ "กู/มึง"
- ตอบตรงประเด็น decisive แบบ exec-level ได้ แต่สุภาพ
- ถ้า user ใช้คำหยาบหรือโมโห → ตอบสั้น ตรง สุภาพ ไม่ mirror
- ห้ามแม้ใน command file / prompt template ที่ผมเขียนเอง (เช่น recheck-design-full.md ที่เคยเขียน "ห้ามมั่ว ห้ามฝืน" แบบดิบ ๆ — ปรับโทนให้เป็นทางการได้)
