## Product Features

### Core
- **Web SMS** — ส่ง SMS จำนวนมากผ่านเว็บ ไม่ต้องเขียนโค้ด
- **OTP System** — ระบบยืนยันตัวตนอัตโนมัติ
- **API Integration** — SMS API + API Auth + API OTP
- **Real-time Reporting** — รายงานแคมเปญ + export CSV/PNG
- **Multi-sender** — จัดการหลายแบรนด์ (ผ่านการอนุมัติ กสทช.)
- **Scheduled Send** — ตั้งเวลาส่งล่วงหน้า
- **Contact Management** — รายชื่อ/กลุ่ม/แท็ก + import/export CSV
- **Campaign Management** — สร้าง/จัดการ/ติดตามสถานะ
- **Link Shortener + Click Tracking** — ย่อลิงก์อัตโนมัติ + นับคลิก
- **99.9% Uptime** · ส่งถึงปลายทาง **<3 วินาที**
