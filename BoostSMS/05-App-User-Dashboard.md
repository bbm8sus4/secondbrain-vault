## หน้าส่ง SMS

- เลือก Sender Name (dropdown)
- ภาษา: **ไทย 70 ตัวอักษร/เครดิต** · **อังกฤษ 160 ตัวอักษร/เครดิต**
- กรอกผู้รับ: พิมพ์เบอร์ / เลือกรายชื่อ / กลุ่ม / อัปโหลด CSV
- เพิ่มลิงก์ (ย่อ URL อัตโนมัติ + นับคลิก)
- Preview ด้านขวา
- สรุปค่าใช้จ่าย: จำนวนผู้รับ × เครดิตต่อข้อความ = เครดิตทั้งหมด
- ตั้งเวลาส่ง หรือส่งทันที

- ทั้งหมด **11 รายชื่อ** (state ปัจจุบัน — เซิร์ฟเวอร์เปิด, แต่ฐานยังเล็ก)
- 3 กลุ่ม: **แบงค์สร้าง · โปรโมชันวันเกิด · ลูกค้าทั่วไป**
- ฟังก์ชัน: นำเข้า, ส่งออก CSV, เพิ่มด่วน, เพิ่มรายชื่อ
- สถานะรายคน: รับ SMS / ถูกบล็อก
