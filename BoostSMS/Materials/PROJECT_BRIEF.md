## Product Features

### Core
- Web SMS: ส่ง SMS จำนวนมากผ่านเว็บ ไม่ต้องเขียนโค้ด
- OTP System: ระบบยืนยันตัวตนอัตโนมัติ
- API Integration: เชื่อมต่อระบบธุรกิจ (SMS API, API Auth, API OTP)
- Real-time Reporting: รายงานแคมเปญแบบละเอียด + export CSV/PNG
- Multi-sender: จัดการหลายแบรนด์ (ผ่านการอนุมัติ กสทช.)
- Scheduled Send: ตั้งเวลาส่งล่วงหน้า
- Contact Management: รายชื่อ, กลุ่ม, แท็ก, import/export CSV
- Campaign Management: สร้าง/จัดการแคมเปญ, ติดตามสถานะ
- Link Shortener + Click Tracking: ย่อลิงก์อัตโนมัติ + นับคลิก
- 99.9% Uptime
- ส่งถึงภายใน 3 วินาที

### Network Coverage
True, DTAC, AIS (ครบทุกค่าย)

### Compliance
NBTC Certified (กสทช.) — Sender Name ต้องผ่านการตรวจสอบ
