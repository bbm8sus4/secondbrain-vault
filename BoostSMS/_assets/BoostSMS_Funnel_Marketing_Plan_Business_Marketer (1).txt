1. เป้าหมายแคมเปญ
แผนนี้เป็นแผนการตลาดใหม่ทั้งหมดสำหรับ BoostSMS โดยจัดตาม Funnel ตั้งแต่การสร้างการรับรู้ ไปจนถึงการปิดการขายและการต่อยอดลูกค้าหลังทดลองใช้ จุดประสงค์หลักคือทำให้เจ้าของธุรกิจและนักการตลาดเห็นว่า “ฐานเบอร์ลูกค้า” ที่มีอยู่สามารถนำกลับมาใช้สร้างยอดขาย แจ้งเตือน และวัดผลได้ผ่าน SMS
* สร้างการรับรู้ว่า BoostSMS คือระบบส่ง SMS สำหรับธุรกิจที่ต้องการเข้าถึงลูกค้าโดยตรง
* ทำให้กลุ่มเป้าหมายเข้าใจว่า SMS ไม่ได้มีไว้แค่ OTP แต่ใช้ทำ Marketing, CRM, Reminder และ Sales Follow-up ได้
* กระตุ้นให้เจ้าของธุรกิจทดลองใช้งานฟรี 100 เครดิต ภายใน 7 วัน
* ดึง Lead เข้าสู่ LINE OA / Inbox / Lead Form และนัด Demo ระบบ
* วางกระบวนการ Sales Follow-up เพื่อเปลี่ยน Trial User ให้เป็น Paid User
* สร้างฐานลูกค้าเริ่มต้นจากกลุ่มธุรกิจที่มีโอกาสซื้อซ้ำและใช้ SMS ต่อเนื่อง
2. กลุ่มเป้าหมายหลักและกลุ่มธุรกิจนำร่อง
แผนนี้ไม่แบ่งกลุ่มเป้าหมายให้มีน้ำหนักเท่ากันทั้งหมด แต่จัดตามบทบาทการตัดสินใจและโอกาสในการใช้งานจริง โดยให้ความสำคัญกับเจ้าของธุรกิจเป็นอันดับแรก เพราะเป็นผู้ตัดสินใจเรื่องงบประมาณและผลลัพธ์ทางธุรกิจ
ระดับ
	กลุ่มเป้าหมาย
	บทบาท
	Message ที่ควรสื่อสาร
	หลัก
	เจ้าของธุรกิจ SME / เจ้าของคลินิก / เจ้าของแบรนด์
	ผู้ตัดสินใจซื้อและอนุมัติงบ
	ใช้ฐานเบอร์ลูกค้าเก่าให้กลับมาเป็นยอดขาย
	รอง
	Marketing Manager / CRM / Sales Manager
	ผู้เสนอเครื่องมือและผู้ใช้งานจริง
	เพิ่มช่องทาง Direct Marketing ที่วัดผลได้
	สนับสนุน
	Admin / Operations / Customer Service
	ผู้ใช้งานระบบประจำวัน
	ลดงานโทรแจ้งลูกค้าและส่งข้อความได้เร็วขึ้น
	

กลุ่มธุรกิจนำร่องที่ควรเริ่มก่อน
• Clinic / Beauty / Wellness - เหมาะที่สุด เพราะมีฐานลูกค้าเก่า การนัดหมายซ้ำ โปรโมชันคอร์ส และมูลค่าต่อลูกค้าสูง
• Retail / E-commerce / Online Seller - เหมาะกับ Flash Sale, โค้ดส่วนลด, แจ้งสถานะออเดอร์ และเรียกลูกค้าเก่ากลับมาซื้อซ้ำ
• Finance / Leasing / Subscription Business - เหมาะกับ Billing Reminder, Payment Link และการลดภาระทีมแอดมิน
• Service Business / Education / Event - ใช้เป็นกลุ่มขยายผลหลังจากได้ Case แรกจากกลุ่มนำร่อง
3. Pain Point & Customer Insight
Insight หลักของแคมเปญคือธุรกิจจำนวนมากมีฐานเบอร์ลูกค้าอยู่แล้ว แต่ยังไม่ได้ใช้ฐานข้อมูลนี้เป็นช่องทางสร้างรายได้อย่างจริงจัง หรือใช้อย่างไม่มีระบบวัดผล
Pain Point
	ผลกระทบต่อธุรกิจ
	BoostSMS ช่วยอย่างไร
	ยิงแอดแพงขึ้น
	ต้นทุนหาลูกค้าใหม่สูงขึ้นทุกเดือน
	ใช้ฐานลูกค้าเดิมส่งโปรโมชันซ้ำ ลดการพึ่งพา Ads อย่างเดียว
	ลูกค้าเก่าหาย
	ยอดขายซ้ำลดลง
	ส่ง SMS Win-back / โปรลูกค้าเก่า / สิทธิ์วันเกิด
	ส่ง LINE หรือโพสต์แล้วลูกค้าไม่เห็น
	Reach ต่ำและข้อความไม่ถึงกลุ่มเป้าหมาย
	SMS เป็นช่องทางตรงถึงมือถือ
	แอดมินต้องโทรแจ้งทีละคน
	เสียเวลาและต้นทุนแรงงาน
	ส่งแจ้งเตือนนัดหมายหรือชำระเงินเป็นชุดได้
	ส่งโปรแล้วไม่รู้ว่าใครสนใจ
	ทีมขายโทรสุ่มและปิดการขายยาก
	ใช้ SMS Tracking ดูว่าใครคลิกลิงก์และคัดเป็น Hot Lead
	

ประโยค Insight ที่ควรใช้ซ้ำในคอนเทนต์: “ฐานเบอร์ลูกค้าที่คุณมี อาจสร้างยอดขายเพิ่มได้ทันที ถ้าใช้ SMS ให้ถูกวิธี”
4. Core Message / Key Selling Points
Core Message หลัก
เปลี่ยนฐานเบอร์ลูกค้าเก่า ให้กลับมาเป็นยอดขาย ด้วย SMS ที่ส่งไว วัดผลได้ และใช้งานง่าย
Message สำหรับยิง Ads
* มีเบอร์ลูกค้าอยู่แล้ว อย่าปล่อยให้เงียบ ส่ง SMS เรียกลูกค้ากลับมาซื้อซ้ำได้ทันที
* ส่งโปรโมชัน แจ้งเตือนนัดหมาย และติดตามลูกค้าได้ในระบบเดียว
* ไม่ต้องเขียนโค้ด มีไฟล์ Excel เบอร์ลูกค้าก็เริ่มส่ง SMS ได้ทันที
* ทดลองใช้ฟรี 100 เครดิต ใช้งานได้ 7 วัน
Key Selling Points
จุดขาย
	คำอธิบายที่ควรใช้
	ใช้งานง่าย
	ส่งผ่าน Web SMS ได้ ไม่ต้องเขียนโค้ด อัปโหลดไฟล์ Excel / CSV แล้วเริ่มส่งได้
	ส่งจำนวนมาก
	รองรับการส่งสูงสุด 10,000 เบอร์ต่อครั้ง
	วัดผลได้
	ใช้ Tracking Link ดูได้ว่าใครคลิก ใครสนใจ และนำไปต่อยอดกับทีมขาย
	ครบทั้ง OTP & API
	รองรับ OTP, 2FA และการเชื่อมต่อระบบผ่าน API
	Dashboard Real-time
	ติดตามผลการส่งและสถิติแคมเปญได้ทันที
	เหมาะกับหลายธุรกิจ
	ใช้ได้กับ Clinic, Retail, E-commerce, Finance, Education และธุรกิจบริการ

6. Marketing Funnel Plan
แผนนี้จัดตาม Funnel เพื่อให้แต่ละกิจกรรมมีหน้าที่ชัดเจน ตั้งแต่ทำให้คนรู้จัก เห็นปัญหา เข้าใจวิธีใช้ ตัดสินใจทดลอง และถูกติดตามจนปิดการขาย

หัวข้อ
	รายละเอียด
	เป้าหมาย

	Message
	คุณมีเบอร์ลูกค้าอยู่แล้ว แต่ใช้มันสร้างยอดขายหรือยัง?
	คอนเทนต์
	Single Post, Short Video, Pain Point Post, Infographic
	ไอเดีย/Action
	ฐานเบอร์ลูกค้าเก่า อาจสร้างยอดขายใหม่ได้ทันที; ยิงแอดแพงขึ้น แต่ลูกค้าเก่าของคุณยังอยู่ในมือ; SMS ยังเป็นช่องทางตรงที่ธุรกิจไม่ควรมองข้าม
