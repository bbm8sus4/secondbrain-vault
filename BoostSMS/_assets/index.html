BoostSMS คืออะไร

BoostSMS เป็นแพลตฟอร์ม SaaS สำหรับส่ง SMS จำนวนมากให้ธุรกิจ เช่น ส่งโปรโมชัน ส่ง OTP ยืนยันตัวตน หรือเชื่อมต่อผ่าน API เข้ากับระบบ CRM/ERP ของลูกค้าโดยตรง

เจ้าของโปรดักส์คือ บริษัท อีซี่สลิป จำกัด (ภายใต้แบรนด์ Thunder Solution) ซื้อ Source Code จากบริษัท มูนรึทึ่ม จำกัด มูลค่า 2 ล้านบาท พัฒนาเพิ่มเติมและเปิดให้ใช้งานตั้งแต่ 15 สิงหาคม 2568
