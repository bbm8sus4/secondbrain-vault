# เงื่อนไขกรอง: ลูกค้าขาดฝาก (Lotto)

## กฎ
ในรายงานยอด Lotto (ไฟล์ Excel `Report_YYYY-MM-DD_HH-MM-SS.xlsx`):

> **ถ้า คอลัมน์ C (ยอดฝาก) = 0  และ  คอลัมน์ S (ยอดสุทธิรวม) = 0**
> → แถวนั้นคือ **ลูกค้าขาดฝาก + ไม่มีการเข้าเล่นในเดือนนั้น**

## ตำแหน่งคอลัมน์อ้างอิง
| คอลัมน์ | ความหมาย                                       |
| ------- | ---------------------------------------------- |
| A       | ชื่อผู้ใช้ (เบอร์)                             |
| B       | สายรหัสสมาชิก (AFF upline)                     |
| **C**   | **ยอดฝาก** ← ใช้กรอง                           |
| D–G     | หวย: ยอดพนัน / AFF / ได้เสีย / สุทธิ           |
| H–K     | Slot & Casino: ยอดพนัน / AFF / ได้เสีย / สุทธิ |
| L–O     | มินิเกม: ยอดพนัน / AFF / ได้เสีย / สุทธิ       |
| P–R     | รวม: ยอดพนัน / AFF / ได้เสีย                   |
| **S**   | **ยอดสุทธิรวม (ทั้งเดือน)** ← ใช้กรอง          |

## วิธีใช้
- **Excel filter:** `C = 0 AND S = 0`
- **Pandas:** `df[(df['ยอดฝาก'] == 0) & (df['ยอดสุทธิรวม'] == 0)]`
- **SQL:** `WHERE deposit = 0 AND net_total = 0`

## Default action เมื่อผู้ใช้ส่งไฟล์ Lotto report มา
**ทุกครั้ง** ที่ผู้ใช้ส่งไฟล์ `Report_*.xlsx` (Lotto) มา:
1. โหลดไฟล์
2. กรองเก็บ **เฉพาะแถวที่ C=0 AND S=0** (ลูกค้าขาดฝาก/ไม่เล่น)
3. **ลบแถวอื่น ๆ ที่ไม่เข้าเงื่อนไขทิ้งทั้งหมด**
4. บันทึกเป็นไฟล์ใหม่ `Report_<วันที่>_inactive_only.xlsx` (เก็บ Cover เดิม + header เดิม)
5. รายงานจำนวน inactive ที่กรองได้ให้ผู้ใช้ทราบ
6. ไม่ต้องถามซ้ำ — ทำให้เลย

## ใช้กับอะไรได้
- ทำ churn list / re-activation campaign (SMS, LINE OA broadcast)
- แยก inactive ออกจากฐาน active เวลาคิด ARPU / win rate จริง
- เช็คสาย AFF ที่มีลูกหลุดเยอะ → upline ต้องดูแล downline
