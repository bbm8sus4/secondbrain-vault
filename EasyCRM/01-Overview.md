# 01 · EasyCRM Overview

## EasyCRM คืออะไร

ระบบบริหารความสัมพันธ์ลูกค้า (CRM) + Loyalty Program — ออกแบบให้ **คล่องตัวและใช้งานง่าย** ตอบโจทย์ธุรกิจขนาดเล็กถึงกลาง

- เก็บข้อมูลลูกค้า + วิเคราะห์พฤติกรรม + ทำการตลาดเฉพาะบุคคล
- เริ่มต้นใช้งานง่าย — เชื่อมต่อกับ LINE Official Account
- มาพร้อมระบบ **ตรวจสลิปโอนเงินอัตโนมัติ** (ผ่าน EasySlip)

## แพลตฟอร์มสองด้าน (Dual Platform)

เชื่อมร้านค้ากับลูกค้าในระบบเดียว ผ่าน LINE OA ของร้าน
- **ฝั่งลูกค้า** — สะสมแต้ม, ดูสิทธิ์, แลกของรางวัล, ใช้งานบน LINE
- **ฝั่งร้าน** — Back Office จัดการลูกค้า/สลิป/คะแนน/คูปอง

## ฟีเจอร์หลัก (Core Capabilities)

| ฟีเจอร์ | คำอธิบาย |
|---|---|
| สะสมคะแนน | สะสมจากการซื้อ เช็คยอด แลกของรางวัลและส่วนลด |
| ระดับสมาชิก | Green · Silver · Gold · Platinum — ยิ่งใช้ยิ่งได้สิทธิ์ |
| ของรางวัล | คูปองส่วนลด ของแถม แลกแต้มสุดคุ้ม |
| สิทธิพิเศษสมาชิก | สิทธิ์เฉพาะสมาชิก กิจกรรมพิเศษ บริการพรีเมียม |
| แคมเปญเฉพาะบุคคล | แบ่งกลุ่มด้วย **RFM** ส่งโปรโมชั่นตรงใจ |
| แดชบอร์ดเรียลไทม์ | ยอดขาย, มูลค่าลูกค้า, ความถี่กลับมา ครบในที่เดียว |

## ตรวจสลิปอัตโนมัติ (จุดขายเด่น)

เหมาะกับธุรกิจ "รับโอนเงินและต้องตรวจสลิปจำนวนมาก" — เชื่อมต่อ **EasySlip**

- ตรวจ: ยอดเงิน · ชื่อบัญชี · เลขบัญชี · เวลาโอน ตรงครบ
- ลดงานคีย์มือ ประหยัดเวลามหาศาล
- บันทึกเข้า EasyCRM ทันที + จ่ายแต้มอัตโนมัติ

## ขั้นตอนสะสมคะแนน (2 รูปแบบ)

**รูปแบบที่ 1 — ตรวจสอบโดยแอดมิน** (ลูกค้าถ่ายรูปใบเสร็จ)
1. ลูกค้าถ่ายรูปใบเสร็จ
2. อัปโหลดเข้าระบบผ่าน LINE OA
3. แอดมินอนุมัติ → ระบบจ่ายแต้มทันที

**รูปแบบที่ 2 — ตรวจสอบอัตโนมัติ** (สลิปโอนเงิน, อัตโนมัติเต็มรูปแบบ)
1. ลูกค้าส่งสลิปโอนเงิน
2. ส่งเข้า LINE OA ที่ใช้ EasyCRM
3. ระบบตรวจสอบผ่าน EasySlip → จ่ายแต้มอัตโนมัติ
