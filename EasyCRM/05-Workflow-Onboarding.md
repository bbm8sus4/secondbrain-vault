# 05 · Workflow & Onboarding

## Flow ลงทะเบียนลูกค้าใหม่ (ครบวงจร)

```
1. กรอกข้อมูลลงทะเบียน
   (ลูกค้ากรอกเอง / Sales กรอกแทน)
        ↓
2. ระบบแจ้งเตือนเข้ากลุ่ม Telegram
   "EasyCRM - SignUp Notify" (อัตโนมัติ)
        ↓
3. ทีม Dev สร้างร้านค้า
   (Subdomain, Portal, ระบบผู้ใช้)
        ↓
4. Dev ตอบกลับ "STORE CONNECTED"
   ในกลุ่มเดิม + ลิงก์ + รหัส
        ↓
5. ส่งมอบให้ Sales
   เริ่มขั้นตอน Onboarding กับลูกค้า
```

## ข้อมูลที่ลูกค้าต้องเตรียม

### สร้างบัญชี
- ชื่อธุรกิจ / ชื่อร้านค้า
- อัตลักษณ์ร้านค้า (สีแบรนด์)
- บัญชี LINE Official Account
- รหัสตัวกลางเชื่อม Messaging API
- ภาพโลโก้
- อีเมล · เบอร์ติดต่อ · ที่อยู่

### การเข้าใช้งานระบบ
- ภาพ Rich Menu
- สร้างทางเข้าใช้งานบน Rich Menu ของ LINE OA
- แนะนำใส่ข้อความ "สมาชิก / สะสมคะแนน"
- เชื่อม Messaging API + LINE Login → ดู [[04-LINE-OA-Setup]]

## 5 Phases การเริ่มต้นใช้งาน (มาตรฐาน 14 วัน)

### Phase 1 — ยืนยันการขาย (วัน 0-1)
- ยืนยันการชำระเงิน
- สร้างประวัติลูกค้า
- ส่งอีเมลต้อนรับ
- แจ้งทีมภายใน

### Phase 2 — วางแผนเริ่มต้น (วัน 1-3)
- นัดประชุม
- คุยเป้าหมาย
- กรอกฟอร์มเริ่มต้น

### Phase 3 — ตั้งค่าระบบ (วัน 5-6)
- ตั้งค่าโปรไฟล์ LINE OA
- ข้อความอัตโนมัติ
- เชื่อมต่อ EasyCRM
- ทดสอบระบบ

### Phase 4 — กราฟิก (วัน 3-11)
- รับโจทย์ออกแบบ
- ออกแบบ Rich Menu
- ลูกค้าอนุมัติ
- ติดตั้งใช้งาน

### Phase 5 — ฝึกอบรม + ส่งมอบ (วัน 10-14)
- เตรียมเอกสาร
- ฝึกอบรม
- ตอบคำถาม
- ลงนามส่งมอบ

## หัวข้อการตั้งค่าระบบ

- ชื่อ + รูปโปรไฟล์ + ภาพปก LINE OA
- ข้อความต้อนรับ + Auto-Reply Keywords
- Rich Menu เริ่มต้น + ระบบป้ายกำกับลูกค้า
- เชื่อมต่อแดชบอร์ด EasyCRM
- เทมเพลตข้อความส่งกลุ่ม
- ทดสอบทุกฟังก์ชั่นก่อนส่งมอบ
