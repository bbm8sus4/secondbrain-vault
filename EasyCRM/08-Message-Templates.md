# 08 · เทมเพลตข้อความติดต่อลูกค้า

> 4 เทมเพลตมาตรฐานสำหรับแอดมินใช้ใน LINE OA

## 1. แจ้งลงทะเบียน

```
ข้อมูลที่ต้องการในการสร้างร้านค้านะคะ

รบกวนกรอกข้อมูลตามแบบฟอร์มหรือแจ้งข้อมูลด้านล่างนี้ค่ะ
https://easycrm-two.vercel.app/register
```

## 2. ขอข้อมูล LINE OA (Messaging API)

```
ข้อมูล Line Official Account (Line OA) ที่ใช้สำหรับการเชื่อมต่อระบบ
รบกวนส่งข้อมูลดังต่อไปนี้ให้แอดมินค่ะ

1. Channel ID :
2. Channel Secret :
3. Access Token :

เมื่อแอดมินได้รับข้อมูลครบถ้วนแล้ว จะดำเนินการสร้างระบบสมาชิกให้ทันทีค่ะ
```
