Product Overview

EasyCRM คืออะไร?

ระบบบริหารความสัมพันธ์ลูกค้าที่ออกแบบมาเพื่อ ความคล่องตัว และ ใช้งานได้ง่าย — ตอบโจทย์ธุรกิจขนาดเล็กถึงกลาง ที่ต้องการระบบที่ไม่ซับซ้อน แต่ครบทุกฟีเจอร์ที่จำเป็น

  • เก็บข้อมูลลูกค้า วิเคราะห์พฤติกรรม และทำการตลาดเฉพาะบุคคล
  • ระบบเริ่มต้นใช้งานได้ง่าย เชื่อมต่อกับ LINE Official Account
  • มาพร้อมระบบตรวจสลิปโอนเงินอัตโนมัติ

2 กรณี การเชื่อมต่อ Messaging API

ขึ้นอยู่กับว่า "ใคร" เป็นคนเปิด Messaging API ของ LINE Official Account ไว้ตั้งแต่แรก

กรณีที่ 1

ยังไม่ได้เปิด Messaging API

ทีมแอดมิน EasyCRM สามารถ เข้าไปตั้งค่าให้ได้ทั้งหมด แล้วโอนสิทธิ์ผู้ให้บริการกลับให้ลูกค้าเมื่อเสร็จเรียบร้อย — ลูกค้าไม่ต้องทำเอง

กรณีที่ 2

ลูกค้าเปิด Messaging API ไว้แล้ว

ลูกค้าต้องเข้า LINE OA Manager และ LINE Developers Console เพื่อดึงค่า Channel ID, Channel Secret, Access Token ด้วยตัวเอง แล้วส่งให้แอดมิน

กรณีพิเศษ — ถ้าลูกค้า ไม่ใช่ผู้สร้างบัญชีผู้ให้บริการเดิม ต้องติดต่อคนที่เปิด Messaging API ตอนแรกเพื่อขอเข้าถึงข้อมูล — ระบบของ LINE ไม่อนุญาตให้คนอื่นเข้าถึงโดยไม่มีสิทธิ์

ขั้นชื่อขั้นตอนรายละเอียดผู้รับผิดชอบระยะเวลา
1ยืนยันการชำระเงินตรวจสอบสลิปหรือใบเสร็จจากลูกค้า บันทึกในระบบฝ่ายการเงินภายใน 2 ชม.
2ส่งอีเมลต้อนรับ + ข้อมูลเข้าระบบส่งอีเมลพร้อมลิงก์เข้าระบบ คู่มือเริ่มต้นผู้บริหารการขายภายใน 24 ชม.
3นัดประชุมเริ่มต้นโครงการวิดีโอคอลแจ้งภาพรวมระบบ และแนะนำทีมงานผู้บริหารการขายภายใน 3 วันทำการ
4ตั้งค่าระบบเริ่มต้นตั้งค่าโปรไฟล์ LINE OA, เมนู, ข้อความอัตโนมัติทีมแอดมินซัปพอร์ตภายใน 3 วันทำการ
5ออกแบบกราฟิกรับโจทย์ ออกแบบ Rich Menu / แบนเนอร์ ตามโควต้านักออกแบบกราฟิก3 วันต่อชิ้นงาน
6ฝึกอบรมการใช้ระบบสอนการใช้งานผ่าน Google Meet หรือสถานที่ลูกค้าผู้บริหารการขายภายใน 5 วันทำการ
7ส่งมอบระบบเริ่มใช้งานจริงทีมแอดมินซัปพอร์ต1 วันหลังตั้งค่าเสร็จ
8ติดตามผลรายเดือนติดตามผล รับข้อเสนอแนะ ตรวจโควต้ากราฟิกทีมแอดมินซัปพอร์ตทุกเดือน (วันที่ 1-5)
9แจ้งเตือนต่ออายุแจ้งเตือนต่ออายุล่วงหน้า 30 วันผู้บริหารการขาย30 วันก่อนหมดอายุ
