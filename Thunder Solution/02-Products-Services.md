## Thunder Corporate API (ขายผ่าน MOU)

แพ็กเกจ Enterprise ที่ขายผ่าน MOU — **ไม่ใช่แพ็กเกจมาตรฐานบนเว็บ**

- ราคาขาย **flat 0.125 บ./สลิป** ทุก tier (ไม่มี tier discount)
- 10 tiers: **Coperate A → J** = 100K → 1M สลิป/เดือน (฿12,500 → ฿125,000/เดือน)
- สัญญาเลือกได้ 1 เดือน หรือ 12 เดือน (volume × 12)
- ต้นทุน 2 scenarios: A (Other-source 100%, cost 0.110, GP 12%) / B (mix blend ~30/70, cost 0.077, GP 38.4%)
- ⚠️ ถ้าใช้ Scenario B **ต้อง lock K-source ≥ 30% ใน MOU** ไม่งั้น margin หาย

รายละเอียดค่าคอม + sales playbook เต็ม → [[03-Revenue-Commission]] และ [[Reports/thunder-corporate-api-sales-commission|Corporate API commission model]]
