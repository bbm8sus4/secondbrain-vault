Thunder Group

พฤษภาคม 2569 (1-22 MTD) เทียบ เมษายน 2569 (เต็มเดือน)

Executive Summary

เดือนพฤษภาคม (22 วัน MTD) ทำรายได้รวม ฿8.07M เทียบกับเมษายนที่ปิดเดือนด้วย ฿10.06M (30 วัน) ถ้าคิดเป็นเฉลี่ยต่อวัน พ.ค. ทำได้ ฿366K/วัน vs เม.ย. ฿335K/วัน — pace ดีขึ้น +9.2% ตัวขับเคลื่อนคือ Thunder API โตระเบิด ฿630K vs ฿514K (+22.5%) ทะลุเป้า 232%

แต่ Thunder Bot ลดลงจาก ฿4.09M (เม.ย.) เหลือ ฿2.85M (พ.ค. MTD) — ยังห่างเป้า ฿4.23M อยู่มาก EasySlip API ก็ลดจาก ฿5.25M เหลือ ฿4.45M แต่ MRR ยังแข็งที่ ฿2.58M Slip margin ปรับดีขึ้นจาก 80.1% (เม.ย.) เป็น 81.7% (พ.ค.) จากต้นทุนต่อสลิปที่ลดลง

Action items: เร่ง Thunder Bot ให้ทำได้อย่างน้อย ฿160K/วัน ใน 9 วันที่เหลือ (ขาดอีก ฿1.4M), push EasySlip API ให้ถึงเป้า ฿5.65M (ขาดอีก ฿1.2M), Boost SMS ยังต้องเพิ่ม ฿17K
